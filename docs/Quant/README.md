<h1 align="center">VibeQuant —— 人人都能学会的量化交易（Vibe Quant）指南</h1>

> **面向零编程基础学习者的量化交易系统化教程，从「我有一个交易想法」到「我构建了一个策略」，让人人都能借助 AI 实现自己的量化梦想。**

点击以访问教程主页：www.vibequant.cn

### 核心理念

践行 **Vibe Coding** 理念——**从 Coder 到 Commander**：通过自然语言与 AI 对话，让量化交易从"写代码"转变为"对话式策略开发"。

> _"完全沉浸于量化交易的'氛围'中，忘记代码的存在。"_

### 快速开始

📖 **在线阅读**：[www.vibequant.cn](https://www.vibequant.cn)

| 你是谁                   | 推荐起点            |
| ------------------------ | ------------------- |
| 完全零基础               | 基础篇 → 第0章 序言 |
| 有 Python 基础但不懂金融 | 基础篇 → 第1-2章    |
| 有交易经验想量化策略     | 进阶篇 → 第3-4章    |
| 想构建完整策略并回测     | 进阶篇 → 第5-6章    |
| 想连接实盘交易           | 实战篇 → 第7-8章    |

### 教程定位

VibeQuant 是国内首个系统化的 Vibe Coding 量化交易开源教程，分为三大板块：

| 板块          | 定位                                  | 适合人群                             |
| ------------- | ------------------------------------- | ------------------------------------ |
| 📘 **基础篇** | Python 编程 + 金融市场基础 + 数据分析 | 完全零基础、学过 Python 但不了解金融 |
| 📗 **策略篇** | 经典策略原理 + Python 实现 + 回测框架 | 有策略想法、需要工具实现验证         |
| 📙 **实战篇** | 实盘对接 + 风险管理 + 组合管理        | 想将策略实盘运行                     |

### 技术栈

- **语言**：Python 3.10+
- **数据**：pandas, numpy, yfinance, akshare
- **策略**：ta-lib, statsmodels, scipy
- **回测**：backtrader, vectorbt, 自定义框架
- **实盘**：alpaca-trade-api, IB API, Binance API

### 学习产出

| 阶段       | 你将获得                                                                 |
| ---------- | ------------------------------------------------------------------------ |
| **基础篇** | 掌握 Python 数据分析能力 · 理解金融市场运作机制 · 能够获取和清洗金融数据 |
| **策略篇** | 理解经典量化策略原理 · 能够用 Python 实现策略 · 掌握回测方法论           |
| **实战篇** | 连接实盘账户 · 实施风险管理 · 构建自己的量化交易系统                     |

```
基础篇目录
│
├── 第0章：序言 —— 量化交易是什么，为什么现在是最好的时代
│   ├── 0.1 量化交易的定义：从主观交易到系统化交易
│   ├── 0.2 量化交易的优势：纪律性、规模化、可验证
│   ├── 0.3 适合人群：谁适合学习量化交易
│   └── 0.4 环境搭建：Python 环境配置与必备库
│
├── 第1章：Python 数据分析基础
│   ├── 1.1 Python 基础回顾：变量、数据结构、函数
│   ├── 1.2 NumPy 数值计算：向量、矩阵、广播
│   ├── 1.3 Pandas 数据处理：Series、DataFrame、时间序列
│   └── 1.4 数据可视化：Matplotlib、Seaborn 基础
│
├── 第2章：金融市场基础
│   ├── 2.1 金融工具：股票、期货、期权、数字货币
│   ├── 2.2 市场结构：交易所、做市商、订单簿
│   ├── 2.3 K线数据：OHLCV 数据结构与含义
│   └── 2.4 订单类型：市价单、限价单、止损单

├── 第3章：金融数据获取与处理
│   ├── 3.1 数据源：yfinance、akshare、Binance API
│   ├── 3.2 数据清洗：缺失值处理、异常值检测
│   ├── 3.3 数据重采样：日线、周线、分钟线转换
│   └── 3.4 财务数据：分红、拆股、财务报表
│
└── 第4章：金融数据分析
    ├── 4.1 收益率计算：对数收益率、简单收益率
    ├── 4.2 分布统计：均值、方差、偏度、峰度
    ├── 4.3 相关性分析：Pearson、Spearman 相关
    └── 4.4 平稳性检验：ADF 检验
```
