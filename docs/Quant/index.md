# VibeQuant

全民 AI 量化学习开源教程。

本教程致力于让每一个人都能学会量化交易，借助 AI 的力量，将自己的交易想法转化为可执行的策略。

## 课程特色

- **零基础友好**：从 Python 基础开始，无需金融背景
- **Vibe Coding 理念**：用自然语言与 AI 对话开发策略
- **全流程覆盖**：数据获取 → 策略开发 → 回测验证 → 实盘交易
- **Python 技术栈**：主流量化开发语言

## 课程体系

### 📘 基础篇

| 章节 | 内容 | 核心交付 |

## 学习路径推荐

```

├── 第3周：数据获取与处理
├── 第4周：金融数据分析

├── 第7周：回测方法论
```
