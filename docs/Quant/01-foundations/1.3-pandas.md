# 1.3 Pandas 数据处理：Series、DataFrame、时间序列

## Pandas 是什么

Pandas 是 Python 中处理表格数据和时间序列的核心库。

在量化交易中，Pandas 用于：

- 存储和管理 OHLCV 金融数据
- 时间序列分析和重采样
- 数据清洗和转换
- 合并多个数据源

## Series

Series 是 Pandas 的基本数据结构，类似一维数组。

### 创建 Series

```python
import pandas as pd

# 从列表创建
s = pd.Series([100, 102, 98, 105])
print(s)
# 0    100
# 1    102
# 2     98
# 3    105
# dtype: int64

# 带索引
s = pd.Series([100, 102, 98, 105], index=['Mon', 'Tue', 'Wed', 'Thu'])
print(s)
# Mon    100
# Tue    102
# Wed     98
# Thu    105
# dtype: int64
```

### Series 索引

```python
s = pd.Series([100, 102, 98, 105], index=['Mon', 'Tue', 'Wed', 'Thu'])

# 位置索引
print(s[0])     # 100
print(s[1:3])  # Tue    102, Wed    98

# 标签索引
print(s['Mon'])      # 100
print(s[['Mon', 'Wed']])  # Mon    100, Wed    98

# 布尔索引
print(s[s > 100])   # Tue    102, Thu    105
```

## DataFrame

DataFrame 是 Pandas 的核心，类似 Excel 表格或 SQL 表。

### 创建 DataFrame

```python
import pandas as pd
import numpy as np

# 从字典创建
data = {
    'AAPL': [100, 102, 98, 105],
    'GOOGL': [2800, 2850, 2780, 2900],
    'MSFT': [300, 305, 298, 310]
}
df = pd.DataFrame(data)
print(df)
#    AAPL  GOOGL  MSFT
# 0   100   2800   300
# 1   102   2850   305
# 2    98   2780   298
# 3   105   2900   310

# 添加日期索引
dates = pd.date_range('2023-01-01', periods=4, freq='D')
df.index = dates
print(df)
#             AAPL  GOOGL  MSFT
# 2023-01-01   100   2800   300
# 2023-01-02   102   2850   305
# 2023-01-03    98   2780   298
# 2023-01-04   105   2900   310
```

### 查看数据

```python
# 头尾数据
print(df.head(2))      # 前2行
print(df.tail(2))      # 后2行

# 基本信息
print(df.info())       # 数据类型、非空数量
print(df.describe())    # 统计摘要（均值、标准差等）

# 行列数
print(df.shape)        # (4, 3)
```

### 选择数据

```python
# 选择列（返回 Series）
print(df['AAPL'])
print(df.AAPL)          # 简写（不建议，有空格或特殊字符时会出错）

# 选择列（返回 DataFrame）
print(df[['AAPL', 'MSFT']])

# 选择行 - 标签
print(df.loc['2023-01-01'])

# 选择行 - 位置
print(df.iloc[0])       # 第一行

# 选择行列（切片）
print(df.loc['2023-01-01':'2023-01-03', ['AAPL', 'MSFT']])
```

### 条件筛选

```python
# 筛选涨幅大于 2% 的股票
df['Return'] = df['AAPL'].pct_change()  # 添加收益率列
print(df[df['Return'] > 0.02])

# 多条件
print(df[(df['AAPL'] > 100) & (df['MSFT'] > 300)])
```

## 时间序列功能

Pandas 提供了强大的时间序列处理能力。

### 日期索引

```python
# 创建日期范围
dates = pd.date_range('2023-01-01', periods=10, freq='D')  # 日
dates = pd.date_range('2023-01-01', periods=12, freq='M')   # 月
dates = pd.date_range('2023-01-01', periods=4, freq='Q')    # 季度

# 按日期筛选
df['2023-01']            # 2023年1月数据
df['2023-01':'2023-03']  # 2023年1-3月数据
```

### 重采样

```python
# 日线转周线
weekly = df.resample('W').agg({
    'Open': 'first',
    'High': 'max',
    'Low': 'min',
    'Close': 'last',
    'Volume': 'sum'
})

# 日线转月线
monthly = df.resample('M').last()

# 滚动窗口
df['MA20'] = df['Close'].rolling(20).mean()      # 20日均线
df['MA60'] = df['Close'].rolling(60).mean()      # 60日均线
df['Vol20'] = df['Close'].rolling(20).std()     # 20日波动率
```

### 移位操作

```python
# 前一天价格
df['Prev_Close'] = df['Close'].shift(1)

# 计算收益率
df['Return'] = df['Close'] / df['Prev_Close'] - 1

# 计算收益（使用 pct_change 更简洁）
df['Return'] = df['Close'].pct_change()
```

### 缺失值处理

```python
# 填充缺失值
df.fillna(method='ffill')  # 用前值填充
df.fillna(method='bfill')   # 用后值填充
df.fillna(0)                # 用0填充

# 删除缺失值
df.dropna()

# 检查缺失值
print(df.isnull().sum())
```

## 数据转换

```python
# 添加计算列
df['Market_Value'] = df['Close'] * df['Volume']

# 条件列
df['Price_Category'] = pd.cut(df['Close'],
                               bins=[0, 100, 200, 300],
                               labels=['Low', 'Medium', 'High'])

# 排名
df['Rank'] = df['Close'].rank(ascending=False)

# 标准化
df['Normalized'] = (df['Close'] - df['Close'].mean()) / df['Close'].std()
```

## 实战：完整的金融数据分析

```python
import pandas as pd
import yfinance as yf
import matplotlib.pyplot as plt

# 1. 获取数据
df = yf.download('AAPL', start='2022-01-01', end='2023-12-31')
df.columns = df.columns.get_level_values(0)

# 2. 计算收益率
df['Daily_Return'] = df['Close'].pct_change()
df['Log_Return'] = np.log(df['Close'] / df['Close'].shift(1))

# 3. 计算滚动指标
df['MA20'] = df['Close'].rolling(20).mean()
df['MA60'] = df['Close'].rolling(60).mean()
df['Vol20'] = df['Daily_Return'].rolling(20).std()

# 4. 输出统计信息
print(df['Daily_Return'].describe())
# count    502.000000
# mean       0.000534
# std        0.024507
# min       -0.082346
# 25%       -0.012078
# 50%        0.001345
# 75%        0.015423
# max        0.088063

# 5. 可视化
fig, axes = plt.subplots(2, 1, figsize=(12, 8))

# 价格和均线
axes[0].plot(df.index, df['Close'], label='Price')
axes[0].plot(df.index, df['MA20'], label='MA20')
axes[0].plot(df.index, df['MA60'], label='MA60')
axes[0].set_title('AAPL Price and Moving Averages')
axes[0].legend()

# 收益率分布
axes[1].hist(df['Daily_Return'].dropna(), bins=50, edgecolor='black')
axes[1].set_title('Daily Return Distribution')

plt.tight_layout()
plt.show()
```
