# 1.1 Python 基础回顾：变量、数据结构、函数

## 为什么需要回顾基础

量化交易开发的代码需要：

- 正确的数据处理
- 清晰的逻辑表达
- 高效的计算

这些都依赖于扎实的 Python 基础。

## 变量与数据类型

### 基本数据类型

```python
# 数值类型
age = 25              # int
price = 99.99         # float
is_trading = True     # bool

# 字符串
ticker = "AAPL"       # str

# 查看类型
print(type(age))       # <class 'int'>
```

### 数据结构

```python
# 列表（List）- 可变序列
prices = [100, 102, 98, 105]
prices.append(103)
print(prices[0])      # 100（索引从0开始）

# 元组（Tuple）- 不可变序列
coordinates = (37.77, 122.41)  # 经纬度

# 字典（Dictionary）- 键值对
stock = {
    'ticker': 'AAPL',
    'price': 178.50,
    'volume': 52000000
}
print(stock['ticker'])  # 'AAPL'

# 集合（Set）- 无序唯一元素
tech_stocks = {'AAPL', 'GOOGL', 'MSFT', 'AAPL'}  # {'AAPL', 'GOOGL', 'MSFT'}
```

## 控制流

### 条件判断

```python
price = 100

if price > 110:
    print("卖出信号")
elif price < 90:
    print("买入信号")
else:
    print("持有")
```

### 循环

```python
# for 循环
prices = [100, 102, 98, 105]
for price in prices:
    print(f"价格: {price}")

# 带索引的循环
for i, price in enumerate(prices):
    print(f"索引 {i}: 价格 {price}")

# while 循环
count = 0
while count < 5:
    print(count)
    count += 1
```

## 函数

函数是组织代码的基本单元。

### 定义函数

```python
def calculate_return(initial, final):
    """计算收益率"""
    return (final - initial) / initial

# 调用函数
ret = calculate_return(100, 110)
print(f"收益率: {ret:.2%}")  # 收益率: 10.00%
```

### 参数默认值

```python
def calculate_return(initial, final, annualize=1):
    """计算收益率，可选择年化"""
    ret = (final - initial) / initial
    return (1 + ret) ** annualize - 1

# 使用默认值
print(calculate_return(100, 110))        # 0.1
# 覆盖默认值
print(calculate_return(100, 110, 252))   # 年化收益率
```

### Lambda 函数

简短函数可以用 lambda 表达式：

```python
# 传统函数
def square(x):
    return x ** 2

# Lambda 函数
square = lambda x: x ** 2

# 常用于排序和筛选
prices = [100, 98, 105, 102]
prices.sort(key=lambda x: -x)  # 降序排列
print(prices)  # [105, 102, 100, 98]
```

## 列表推导式

列表推导式是 Python 的特色功能，可以让代码更简洁：

```python
# 传统方式
squares = []
for i in range(10):
    squares.append(i ** 2)

# 列表推导式
squares = [i ** 2 for i in range(10)]

# 带条件
even_squares = [i ** 2 for i in range(10) if i % 2 == 0]
print(even_squares)  # [0, 4, 16, 36, 64]
```

## 异常处理

健壮的代码需要处理异常：

```python
def get_price(stock_dict, key):
    try:
        return stock_dict[key]
    except KeyError:
        return None

stock = {'AAPL': 178.50, 'GOOGL': 142.30}
print(get_price(stock, 'MSFT'))  # None（不存在返回None）
```

## 模块导入

```python
# 导入整个模块
import math
print(math.sqrt(16))  # 4.0

import numpy as np
print(np.mean([1, 2, 3, 4, 5]))  # 3.0
```

## 实践练习

### 练习 1：计算多个股票的收益率

```python
# 给定数据
stocks = {
    'AAPL': {'buy': 150, 'sell': 175},
    'GOOGL': {'buy': 2800, 'sell': 2950},
    'MSFT': {'buy': 300, 'sell': 325}
}

# 目标：计算每只股票的收益率，并找出收益率最高的股票
```

<details>
<summary><b>参考答案</b></summary>

```python
stocks = {
    'AAPL': {'buy': 150, 'sell': 175},
    'GOOGL': {'buy': 2800, 'sell': 2950},
    'MSFT': {'buy': 300, 'sell': 325}
}

returns = {}
for ticker, prices in stocks.items():
    ret = (prices['sell'] - prices['buy']) / prices['buy']
    returns[ticker] = ret

best_stock = max(returns, key=returns.get)
print(f"收益率: {returns}")
print(f"最佳股票: {best_stock}, 收益率: {returns[best_stock]:.2%}")
```
