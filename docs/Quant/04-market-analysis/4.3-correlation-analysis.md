# 4.3 相关性分析：Pearson、Spearman相关性

## 相关性的重要性

```python
# 相关性是量化交易的核心概念
# 1. 分散化：低相关性资产组合可以降低风险
# 2. 配对交易：利用高相关性资产的价差
# 3. 风险管理：了解资产间的关联性

import numpy as np
import pandas as pd
import yfinance as yf
```

## Pearson 相关系数

### 定义

```python
# Pearson相关系数衡量线性关系
# 范围：-1 到 +1

def pearson_correlation(x, y):
    """手动计算Pearson相关系数"""
    n = len(x)
    mean_x = np.mean(x)
    mean_y = np.mean(y)

    numerator = np.sum((x - mean_x) * (y - mean_y))
    denominator = np.sqrt(np.sum((x - mean_x)**2) * np.sum((y - mean_y)**2))

    return numerator / denominator

# 或者使用pandas
# returns['AAPL'].corr(returns['GOOGL'])
```

### 计算示例

```python
# 获取多只股票
stocks = ['AAPL', 'GOOGL', 'MSFT', 'AMZN']
data = yf.download(stocks, start='2022-01-01', end='2023-12-31')['Close']
data.columns = data.columns.get_level_values(0)

# 计算收益率
returns = data.pct_change().dropna()

# Pearson相关矩阵
pearson_corr = returns.corr(method='pearson')
print("Pearson相关矩阵:")
print(pearson_corr.round(3))
```

## Spearman 相关系数

### 定义

```python
# Spearman相关系数衡量单调关系（不限于线性）
# 基于秩次（rank），对异常值更鲁棒

def spearman_correlation(x, y):
    """手动计算Spearman相关系数"""
    from scipy.stats import spearmanr
    corr, _ = spearmanr(x, y)
    return corr

# 或者使用pandas
# returns['AAPL'].corr(returns['GOOGL'], method='spearman')
```

### 计算示例

```python
# Spearman相关矩阵
spearman_corr = returns.corr(method='spearman')
print("Spearman相关矩阵:")
print(spearman_corr.round(3))

# 比较Pearson和Spearman
print("\n差异 (Pearson - Spearman):")
print((pearson_corr - spearman_corr).round(3))
```

## 相关性的解读

```python
# 相关性强度解读
# |r| < 0.3: 弱相关
# 0.3 <= |r| < 0.5: 中等相关
# 0.5 <= |r| < 0.7: 较强相关
# |r| >= 0.7: 强相关

def interpret_correlation(r):
    """解读相关性"""
    abs_r = abs(r)
    if abs_r < 0.3:
        return "弱"
    elif abs_r < 0.5:
        return "中等"
    elif abs_r < 0.7:
        return "较强"
    else:
        return "强"
```

fig, axes = plt.subplots(1, 2, figsize=(14, 5))

## 时间变化的相关性

```python
# 计算滚动相关性
rolling_corr = returns['AAPL'].rolling(60).corr(returns['GOOGL'])

# 可视化
plt.figure(figsize=(12, 5))
plt.plot(rolling_corr.index, rolling_corr)
plt.axhline(rolling_corr.mean(), color='red', linestyle='--', label=f'Mean: {rolling_corr.mean():.3f}')
plt.title('60-Day Rolling Correlation: AAPL vs GOOGL')
plt.xlabel('Date')
plt.ylabel('Correlation')
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

# 相关性变化的原因
# 1. 市场状态变化（牛市vs熊市）
# 2. 行业周期
# 3. 宏观经济因素
```

## 相关性的陷阱

```python
# 1. 虚假相关
# 两个不相关的变量可能因为共同趋势显示高相关

# 2. 幸存者偏差
# 只用现存股票会高估相关性

# 3. 期限错配
# 用日数据和月数据计算相关性可能不准确

# 4. 非线性关系
# Pearson=0不代表没有关系，可能是非线性
```
