# 第 3 章：数据获取与处理

## 本章概述

数据是量化交易的基础。本章将教你如何获取、清洗和处理金融数据。

## 学习目标

完成本章学习后，你将能够：

1. ✅ 使用多种数据源获取金融数据
2. ✅ 清洗和处理金融数据
3. ✅ 处理不同频率的时间序列数据
4. ✅ 获取和处理财务数据
