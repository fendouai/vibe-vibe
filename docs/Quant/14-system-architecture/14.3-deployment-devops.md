# 14.3 部署与DevOps

## 部署架构

量化交易系统的部署需要考虑：

- **可靠性**: 系统必须稳定运行
- **隔离性**: 不同策略之间隔离
- **可观测性**: 日志、监控、告警
- **快速恢复**: 故障时快速切换

## Docker容器化

### Dockerfile

```dockerfile
FROM python:3.11-slim

# 设置工作目录
WORKDIR /app

# 安装系统依赖
RUN apt-get update && apt-get install -y \
    build-essential \
    curl \
    && rm -rf /var/lib/apt/lists/*

# 复制依赖文件
COPY requirements.txt .

# 安装Python依赖
RUN pip install --no-cache-dir -r requirements.txt

# 复制代码
COPY . .

# 创建非root用户
RUN useradd -m appuser && chown -R appuser:appuser /app
USER appuser

# 健康检查
HEALTHCHECK --interval=30s --timeout=10s --start-period=5s --retries=3 \
    CMD curl -f http://localhost:8080/health || exit 1

# 启动命令
CMD ["python", "src/main.py"]
```

import os

## 持续集成/持续部署 (CI/CD)

### GitHub Actions

```yaml
# .github/workflows/deploy.yml
name: Deploy Trading System

on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3

      - name: Set up Python
        uses: actions/setup-python@v4
        with:
          python-version: '3.11'

      - name: Install dependencies
        run: |
          pip install -r requirements.txt
          pip install pytest pytest-cov

      - name: Run tests
        run: pytest tests/ --cov=src

      - name: Lint
        run: |
          pip install flake8 black
          flake8 src
          black --check src

  build:
    needs: test
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3

      - name: Build Docker image
        run: docker build -t trading-system:${{ github.sha }} .

      - name: Run integration tests
        run: docker-compose -f docker-compose.test.yml up --abort-on-container-exit

      - name: Push to registry
        if: github.ref == 'refs/heads/main'
        run: |
          echo "${{ secrets.DOCKER_TOKEN }}" | docker login -u "${{ secrets.DOCKER_USERNAME }}" --password-stdin
          docker tag trading-system:${{ github.sha }} registry.example.com/trading-system:latest
          docker push registry.example.com/trading-system:latest

  deploy:
    needs: build
    if: github.ref == 'refs/heads/main'
    runs-on: ubuntu-latest
    steps:
      - name: Deploy to server
        uses: appleboy/ssh-action@master
        with:
          host: ${{ secrets.SERVER_HOST }}
          username: deploy
          key: ${{ secrets.SERVER_SSH_KEY }}
          script: |
            cd /opt/trading-system
            docker-compose pull
            docker-compose up -d
            docker-compose exec -T trading_system python src/health_check.py
```

## 服务器部署

### Ubuntu服务器部署脚本

```bash
#!/bin/bash
# deploy.sh - 生产环境部署脚本

set -e

# 配置
APP_DIR="/opt/trading-system"
BACKUP_DIR="/opt/backups"
DATE=$(date +%Y%m%d_%H%M%S)

echo "=== 开始部署 ==="

# 1. 备份
echo "1. 备份当前版本..."
mkdir -p $BACKUP_DIR
cp -r $APP_DIR $BACKUP_DIR/trading-system-$DATE

# 2. 拉取最新代码
echo "2. 拉取最新代码..."
cd $APP_DIR
git pull origin main

# 3. 构建Docker镜像
echo "3. 构建Docker镜像..."
docker build -t trading-system:latest .

# 4. 重启服务
echo "4. 重启服务..."
docker-compose down
docker-compose up -d

# 5. 健康检查
echo "5. 健康检查..."
sleep 10
curl -f http://localhost:8080/health || {
    echo "健康检查失败，回滚..."
    docker-compose down
    docker-compose -f $BACKUP_DIR/trading-system-$DATE/docker-compose.yml up -d
    exit 1
}

# 6. 清理旧备份（保留最近5个）
echo "6. 清理旧备份..."
ls -dt $BACKUP_DIR/trading-system-* | tail -n +6 | xargs rm -rf

echo "=== 部署完成 ==="
```

### 自定义指标

```python
from prometheus_client import Counter, Gauge, Histogram, start_http_server

# 定义指标
orders_placed = Counter('orders_placed_total', 'Total orders placed', ['symbol', 'side'])
orders_filled = Counter('orders_filled_total', 'Total orders filled', ['symbol'])
order_latency = Histogram('order_latency_seconds', 'Order processing latency')
portfolio_value = Gauge('portfolio_value', 'Current portfolio value')
position_size = Gauge('position_size', 'Current position size', ['symbol'])

# 在交易时记录
def place_order(symbol, quantity, side):
    start = time.time()
    try:
        order = broker.submit_order(symbol, quantity, side)
        orders_placed.labels(symbol=symbol, side=side).inc()
        return order
    finally:
        order_latency.observe(time.time() - start)

# 启动HTTP服务器提供metrics
start_http_server(8080)
```

### 告警规则

```yaml
# alert_rules.yml
groups:
  - name: trading_alerts
    rules:
      - alert: HighOrderRejectionRate
        expr: rate(orders_rejected_total[5m]) / rate(orders_placed_total[5m]) > 0.1
        for: 5m
        labels:
          severity: critical
        annotations:
          summary: '订单拒绝率超过10%'

      - alert: LargeDrawdown
        expr: portfolio_drawdown > 0.1
        for: 1m
        labels:
          severity: critical
        annotations:
          summary: '组合回撤超过10%'

      - alert: SystemDown
        expr: up{job="trading_system"} == 0
        for: 1m
        labels:
          severity: critical
        annotations:
          summary: '交易系统不可用'
```

## 备份和恢复

```bash
#!/bin/bash
# backup.sh - 数据库备份脚本

BACKUP_DIR="/opt/backups"
DATE=$(date +%Y%m%d_%H%M%S)

# PostgreSQL备份
pg_dump -Fc trading_prod > $BACKUP_DIR/db_$DATE.dump

# Redis备份
redis-cli SAVE

# 备份到远程
aws s3 cp $BACKUP_DIR/db_$DATE.dump s3://trading-backups/

# 保留最近30个备份
ls -dt $BACKUP_DIR/db_* | tail -n +31 | xargs rm -f
```
