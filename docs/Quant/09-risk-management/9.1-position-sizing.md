# 9.1 头寸管理：Kelly公式、固定比例

## 头寸管理公式

```python
"""
头寸管理 (Position Sizing)：

决定每笔交易应该投入多少资金。

核心问题：
- 投入太少：收益低
- 投入太多：风险大

方法：
1. Kelly公式
2. 固定比例
3. 波动率调整
"""
```

## Kelly公式

```python
def kelly_criterion(win_rate, avg_win, avg_loss):
    """
    Kelly公式计算最优仓位比例

    Kelly% = W - (1-W)/R
    其中：
    - W = 胜率
    - R = 盈亏比

    注意：
    - Kelly公式给出理论最优
    - 实际使用建议 Kelly/2（半Kelly）降低波动
    """
    if avg_loss == 0:
        return 0

    win_prob = win_rate
    loss_prob = 1 - win_rate
    reward_risk = avg_win / avg_loss

    kelly_pct = (win_prob * reward_risk - loss_prob) / reward_risk

    # 限制在合理范围
    kelly_pct = max(0, min(kelly_pct, 0.5))  # 最大50%

    return kelly_pct

# 示例
win_rate = 0.55
avg_win = 0.10
avg_loss = 0.05

kelly = kelly_criterion(win_rate, avg_win, avg_loss)
print(f"Kelly仓位: {kelly:.2%}")
print(f"半Kelly仓位: {kelly/2:.2%}")
```

## 固定比例法

```python
def fixed_fraction(position_size, account_value, risk_per_trade=0.02):
    """
    固定比例法

    每笔交易承担固定比例的账户风险
    """
    return account_value * risk_per_trade / position_size

def fixed_ratio(account_value, initial_balance, delta=1000):
    """
    固定比例法（基于账户增长）

    每增加固定金额，增加一个单位头寸
    """
    n_units = int((account_value - initial_balance) / delta)
    return max(1, n_units)
```
