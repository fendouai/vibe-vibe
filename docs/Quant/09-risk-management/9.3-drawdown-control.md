# 9.3 最大回撤控制：动态仓位调整

## 回撤控制策略

```python
"""
最大回撤控制：

当账户从历史高点回撤超过阈值时，
减小仓位或停止交易。

方法：
1. 固定回撤阈值
2. 动态仓位调整
3. 分阶段减仓
"""
```

## 回撤控制实现

```python
def drawdown_control(signals, returns, max_drawdown=0.15, reduce_factor=0.5):
    """
    回撤控制

    当回撤超过阈值时，减小仓位

    Args:
        signals: 原始信号
        returns: 收益率
        max_drawdown: 最大允许回撤
        reduce_factor: 减仓系数
    """
    adjusted_signals = signals.copy().astype(float)

    wealth = (1 + returns).cumprod()
    peak = wealth.cummax()
    drawdown = (wealth - peak) / peak

    # 如果当前回撤超过阈值
    current_dd = drawdown.iloc[-1]

    if current_dd < -max_drawdown:
        # 减仓
        adjusted_signals = signals * reduce_factor

    return adjusted_signals

class DrawdownRiskManager:
    """回撤风险管理器"""

    def __init__(self, initial_capital, max_drawdown=0.15, reduce_factor=0.5):
        self.initial_capital = initial_capital
        self.max_drawdown = max_drawdown
        self.reduce_factor = reduce_factor
        self.peak_capital = initial_capital
        self.current_capital = initial_capital
        self.position_size = 1.0

    def update(self, pnl):
        """更新资本和回撤"""
        self.current_capital += pnl
        self.peak_capital = max(self.peak_capital, self.current_capital)

        drawdown = (self.current_capital - self.peak_capital) / self.peak_capital

        if drawdown < -self.max_drawdown:
            self.position_size *= self.reduce_factor
            print(f"回撤超过阈值，减仓至 {self.position_size:.0%}")

    def get_position_size(self):
        """获取当前仓位系数"""
        return self.position_size
```
