# 9.2 止损策略：固定止损、时间止损

## 止损类型

```python
"""
止损 (Stop Loss)：

是风险管理的最后防线。

类型：
1. 固定止损 - 价格达到特定水平止损
2. 时间止损 - 持有特定时间后止损
3. 追踪止损 - 价格向有利方向移动后调整止损
"""
```

## 固定止损

```python
def calculate_stop_loss(entry_price, stop_pct=0.05, side='long'):
    """
    计算固定止损价格

    Args:
        entry_price: 入场价格
        stop_pct: 止损百分比
        side: 'long' 或 'short'
    """
    if side == 'long':
        stop_price = entry_price * (1 - stop_pct)
    else:
        stop_price = entry_price * (1 + stop_pct)

    return stop_price

# 示例
entry = 100
stop = calculate_stop_loss(entry, stop_pct=0.05, side='long')
print(f"止损价格: {stop:.2f}")
```

## 追踪止损

```python
def trailing_stop(current_price, peak_price, stop_pct=0.10, side='long'):
    """
    追踪止损

    止损价格随价格向有利方向移动而调整
    """
    if side == 'long':
        stop_price = peak_price * (1 - stop_pct)
    else:
        stop_price = peak_price * (1 + stop_pct)

    return stop_price

# 示例
peak_price = 110  # 持仓期间最高价
current = 115
stop = trailing_stop(current, peak_price, stop_pct=0.10)
print(f"追踪止损: {stop:.2f}")
```
