# 6.2 布林带策略：价格通道与买卖信号

## 布林带原理

```python
"""
布林带 (Bollinger Bands)：

由John Bollinger发明，是最常用的均值回归指标之一。

组成：
- 中轨：N日简单移动平均 (SMA)
- 上轨：中轨 + K × 标准差
- 下轨：中轨 - K × 标准差

默认参数：N=20, K=2

信号逻辑：
- 价格触及上轨 → 可能反转向下（卖出）
- 价格触及下轨 → 可能反转向上（买入）
"""

bollinger_bands = """
         价格
          ↑
     上轨 ─ ─ ─ ─ ─ ─ ─ ─ ─ → 卖出信号
          │   ↑    ↑
   中轨 ─ ─┼─── ← ─┼──→ 回归
          │   ↓    ↓
     下轨 ─ ─ ─ ─ ─ ─ ─ ─ ─ → 买入信号
          └──────────────────→ 时间
"""
```

## Python 实现

```python
import pandas as pd
import numpy as np
import yfinance as yf

def bollinger_bands(prices, window=20, num_std=2):
    """
    计算布林带

    Args:
        prices: 价格序列
        window: 移动平均窗口
        num_std: 标准差倍数

    Returns:
        DataFrame with BBands
    """
    bb = pd.DataFrame(index=prices.index)

    bb['middle'] = prices.rolling(window).mean()
    bb['std'] = prices.rolling(window).std()
    bb['upper'] = bb['middle'] + num_std * bb['std']
    bb['lower'] = bb['middle'] - num_std * bb['std']

    return bb

# 计算布林带
df = yf.download('AAPL', start='2022-01-01', end='2023-12-31')
df.columns = df.columns.get_level_values(0)
prices = df['close']

bb = bollinger_bands(prices, window=20, num_std=2)

# 可视化
plt.figure(figsize=(12, 6))
plt.plot(prices, label='Price', linewidth=1.5)
plt.plot(bb['middle'], label='Middle Band (MA20)', alpha=0.7)
plt.plot(bb['upper'], label='Upper Band', linestyle='--', alpha=0.7)
plt.plot(bb['lower'], label='Lower Band', linestyle='--', alpha=0.7)
plt.fill_between(bb.index, bb['upper'], bb['lower'], alpha=0.1)
plt.title('Bollinger Bands')
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()
```

## 布林带交易策略

```python
def bollinger_strategy(prices, window=20, num_std=2, exit_pct=0.5):
    """
    布林带交易策略

    策略规则：
    1. 买入信号：价格触及下轨
    2. 卖出信号：价格触及上轨 或 回归到中轨的50%位置

    Args:
        prices: 价格序列
        window: 窗口
        num_std: 标准差倍数
        exit_pct: 回归到中轨的百分比

    Returns:
        signals: 交易信号
    """
    bb = bollinger_bands(prices, window, num_std)

    signals = pd.Series(0, index=prices.index)

    # 买入信号：价格触及下轨
    buy_signal = prices < bb['lower']
    signals[buy_signal] = 1

    # 卖出信号：价格触及上轨
    sell_signal = prices > bb['upper']
    signals[sell_signal] = -1

    # 或者用中轨作为动态止盈/止损
    middle_reached = (prices - bb['middle']).abs() < prices * 0.01
    signals[middle_reached & (signals == 1)] = 0  # 多头平仓

    return signals.shift(1).fillna(0)

# 测试策略
signals = bollinger_strategy(prices)
print(f"买入信号次数: {(signals == 1).sum()}")
print(f"卖出信号次数: {(signals == -1).sum()}")
```
