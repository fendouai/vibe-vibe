# 6.1 均值回归理论：均值回归的原理和数学基础

## 均值回归的逻辑

```python
"""
均值回归的核心理念：

1. 价值假设
   - 每个资产都有"内在价值"
   - 市场价格会围绕价值波动

2. 偏离与回归
   - 当价格显著高于均值 → 卖出
   - 当价格显著低于均值 → 买入
   - 价格终将回归

3. 均值类型
   - 时间均值（移动平均）
   - 统计均值（数学期望）
   - 价值均值（基本面估值）
"""
```

## Ornstein-Uhlenbeck 过程

```python
"""
OU过程是描述均值回归的经典随机微分方程：

dX = θ(μ - X)dt + σdW

其中：
- X: 价格或价差
- μ: 均值水平
- θ: 回归速度（越大回归越快）
- σ: 波动率
- dW: 布朗运动

离散形式：
X_{t+1} = X_t + θ(μ - X_t) + ε

这意味着：
- 如果 X_t > μ，X_{t+1} < X_t （倾向于下跌）
- 如果 X_t < μ，X_{t+1} > X_t （倾向于上涨）
"""

import numpy as np

def ou_process_simulation(theta, mu, sigma, x0, n_steps, dt=1):
    """
    模拟Ornstein-Uhlenbeck过程

    Args:
        theta: 回归速度
        mu: 均值水平
        sigma: 波动率
        x0: 初始值
        n_steps: 模拟步数
        dt: 时间步长

    Returns:
        模拟的价格序列
    """
    x = np.zeros(n_steps)
    x[0] = x0

    for t in range(1, n_steps):
        x[t] = x[t-1] + theta * (mu - x[t-1]) + sigma * np.random.randn() * np.sqrt(dt)

    return x

# 示例
theta = 0.5   # 回归速度
mu = 100     # 均值
sigma = 2    # 波动率
x0 = 120     # 初始价格（高于均值）

prices = ou_process_simulation(theta, mu, sigma, x0, 500)

import matplotlib.pyplot as plt
plt.figure(figsize=(12, 5))
plt.plot(prices)
plt.axhline(mu, color='red', linestyle='--', label=f'Mean = {mu}')
plt.title('Ornstein-Uhlenbeck Process (Mean Reversion)')
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()
```

## 均值回归的关键参数

```python
"""
1. 回看期 (Lookback Period)
   - 计算均值的窗口大小
   - 太短：噪声多，假信号
   - 太长：反应迟钝，错过机会

2. 回归阈值 (Reversion Threshold)
   - 价格偏离多少才触发信号
   - 常用：1, 1.5, 2 倍标准差

3. 持仓期 (Holding Period)
   - 回归到均值需要的时间
   - 持有太短：没等到回归就平仓
   - 持有太长：可能从回归变成再次偏离

4. 止损 (Stop Loss)
   - 如果价格继续偏离，需要止损
   - 设置最大亏损限制
"""
```

## Z-Score

```python
"""
Z-Score 标准化价格偏离程度：

Z = (X - μ) / σ

含义：
- Z = 0：价格等于均值
- Z = 2：价格偏离均值2个标准差
- Z = -2：价格低于均值2个标准差

使用：
- |Z| > 2：极端偏离，可能反转
- |Z| < 0.5：接近均值
"""

def calculate_zscore(series, window=20):
    """计算Z-Score"""
    mean = series.rolling(window).mean()
    std = series.rolling(window).std()
    zscore = (series - mean) / std
    return zscore

# 示例
df = yf.download('AAPL', start='2022-01-01', end='2023-12-31')
df.columns = df.columns.get_level_values(0)
prices = df['close']

plt.figure(figsize=(12, 5))
plt.plot(zscore)
plt.axhline(2, color='red', linestyle='--', label='Upper Threshold (2)')
plt.axhline(-2, color='green', linestyle='--', label='Lower Threshold (-2)')
plt.axhline(0, color='gray', linestyle='-', alpha=0.5)
plt.title('Z-Score of AAPL Price')
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()
```
