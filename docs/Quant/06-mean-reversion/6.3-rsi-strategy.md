# 6.3 RSI策略：超买超卖指标应用

## RSI原理

```python
"""
RSI (Relative Strength Index) 相对强弱指数：

由J. Welles Wilder Jr.发明，是最常用的动量/均值回归指标。

计算公式：
RSI = 100 - (100 / (1 + RS))

其中：
RS = 平均涨幅 / 平均跌幅

参数：通常使用14天

解读：
- RSI > 70：超买区域，可能反转向下
- RSI < 30：超卖区域，可能反转向上
- RSI > 80：极端超买
- RSI < 20：极端超卖
"""
```

## Python 实现

```python
def calculate_rsi(prices, period=14):
    """
    计算RSI

    Args:
        prices: 价格序列
        period: RSI周期

    Returns:
        RSI序列
    """
    delta = prices.diff()

    gain = delta.where(delta > 0, 0).rolling(period).mean()
    loss = (-delta.where(delta < 0, 0)).rolling(period).mean()

    rs = gain / loss
    rsi = 100 - (100 / (1 + rs))

    return rsi

# 计算RSI
rsi = calculate_rsi(prices, period=14)

# 可视化
fig, axes = plt.subplots(2, 1, figsize=(12, 8), gridspec_kw={'height_ratios': [2, 1]})

axes[0].plot(prices)
axes[0].set_title('AAPL Price')

axes[1].plot(rsi)
axes[1].axhline(70, color='red', linestyle='--', label='Overbought (70)')
axes[1].axhline(30, color='green', linestyle='--', label='Oversold (30)')
axes[1].axhline(50, color='gray', linestyle='-', alpha=0.5)
axes[1].fill_between(rsi.index, 70, 100, alpha=0.1, color='red')
axes[1].fill_between(rsi.index, 0, 30, alpha=0.1, color='green')
axes[1].set_title('RSI (14)')
axes[1].set_ylim(0, 100)
axes[1].legend()
axes[1].grid(True, alpha=0.3)

plt.tight_layout()
plt.show()
```

## RSI 交易策略

```python
def rsi_strategy(prices, period=14, buy_threshold=30, sell_threshold=70):
    """
    RSI交易策略

    策略规则：
    - RSI < buy_threshold → 买入（超卖）
    - RSI > sell_threshold → 卖出（超买）
    """
    rsi = calculate_rsi(prices, period)

    signals = pd.Series(0, index=prices.index)

    # 买入信号
    buy = rsi < buy_threshold
    signals[buy] = 1

    # 卖出信号
    sell = rsi > sell_threshold
    signals[sell] = -1

    return signals.shift(1).fillna(0)

# 测试
signals = rsi_strategy(prices)
print(f"买入信号: {(signals == 1).sum()}")
print(f"卖出信号: {(signals == -1).sum()}")
```
