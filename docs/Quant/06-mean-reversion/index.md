# 第 6 章：均值回归策略 (Mean Reversion)

## 均值回归理论

均值回归是金融市场中的另一个经典现象：**价格偏离均值后会回归**。

```python
"""
均值回归策略：

核心假设：
1. 价格围绕价值（均值）波动
2. 偏离越大，回归概率越高
3. 回归时间不确定

适用场景：震荡市场
注意：趋势市场可能失效
"""

mean_reversion_principle = """
         价格
          ↑
     上轨 ─ ─ ─ ─ ─ ─ → 卖出信号
          │    ↑
   均值 ─ ─┼──→ 回归到均值
          │    ↓
     下轨 ─ ─ ─ ─ ─ ─ → 买入信号
          └──────────────→ 时间
"""
```
