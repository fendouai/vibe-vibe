# 10.1 现代投资组合理论 (MPT)

## 什么是现代投资组合理论？

现代投资组合理论（Modern Portfolio Theory, MPT）由哈里·马科维茨（Harry Markowitz）在1952年提出，是量化投资领域的基石理论。MPT的核心思想是：**不要把鸡蛋放在一个篮子里**——通过组合多个不相关或负相关的资产，可以在不降低预期收益的情况下降低投资组合的整体风险。

## 核心概念

### 1. 预期收益 (Expected Return)

$$E(R_p) = \sum_{i} w_i \cdot E(R_i)$$

其中 $w_i$ 是资产 $i$ 的权重，$E(R_i)$ 是资产 $i$ 的预期收益。

### 2. 风险 (方差/标准差)

$$\sigma_p^2 = \sum_{i} \sum_{j} w_i \cdot w_j \cdot \sigma_{ij}$$

当 $i=j$ 时，$\sigma_{ij}$ 是资产 $i$ 的方差。

### 3. 协方差与相关系数

$$\sigma_{ij} = \rho_{ij} \cdot \sigma_i \cdot \sigma_j$$

其中 $\rho_{ij}$ 是资产 $i$ 和 $j$ 之间的相关系数。

## Python实现

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from scipy.optimize import minimize

# 假设我们有3个策略的历史收益
np.random.seed(42)
n_assets = 3
n_days = 252

# 生成模拟数据（实际使用时用真实历史数据）
returns = pd.DataFrame({
    'Momentum': np.random.randn(n_days) * 0.02 + 0.0003,
    'Mean_Reversion': np.random.randn(n_days) * 0.015 + 0.00025,
    'Pairs': np.random.randn(n_days) * 0.01 + 0.0002
})

# 计算平均收益和协方差矩阵
mean_returns = returns.mean()
cov_matrix = returns.cov()

print("平均日收益:")
print(mean_returns)
print("\n协方差矩阵:")
print(cov_matrix)
```

## 有效前沿 (Efficient Frontier)

有效前沿是所有最优投资组合的集合——在这些组合中，给定风险水平下收益最高，或给定收益水平下风险最低。

```python
def portfolio_return(weights, mean_returns):
    """计算组合收益"""
    return np.sum(weights * mean_returns)

def portfolio_volatility(weights, cov_matrix):
    """计算组合波动率"""
    return np.sqrt(np.dot(weights.T, np.dot(cov_matrix, weights)))

def neg_sharpe_ratio(weights, mean_returns, cov_matrix, risk_free_rate=0.02/252):
    """负夏普比率（用于最小化）"""
    p_ret = portfolio_return(weights, mean_returns)
    p_vol = portfolio_volatility(weights, cov_matrix)
    return -(p_ret - risk_free_rate) / p_vol

def minimize_volatility(weights, mean_returns, cov_matrix):
    """最小化波动率"""
    return portfolio_volatility(weights, cov_matrix)

# 设置优化问题
num_portfolios = 1000
risk_free_rate = 0.02 / 252

# 存储结果
results = np.zeros((3, num_portfolios))
weights_record = []

for i in range(num_portfolios):
    # 随机生成权重
    weights = np.random.random(n_assets)
    weights /= np.sum(weights)
    weights_record.append(weights)

    # 计算组合表现
    portfolio_return_i = portfolio_return(weights, mean_returns)
    portfolio_volatility_i = portfolio_volatility(weights, cov_matrix)

    results[0, i] = portfolio_volatility_i
    results[1, i] = portfolio_return_i
    results[2, i] = (portfolio_return_i - risk_free_rate) / portfolio_volatility_i

# 找到最优夏普比率组合
max_sharpe_idx = results[2].argmax()
max_sharpe_weights = weights_record[max_sharpe_idx]

# 找到最小方差组合
min_vol_idx = results[0].argmax()
min_vol_weights = weights_record[min_vol_idx]

print(f"最优夏普比率组合:")
print(f"  权重: Momentum={max_sharpe_weights[0]:.3f}, Mean_Reversion={max_sharpe_weights[1]:.3f}, Pairs={max_sharpe_weights[2]:.3f}")
print(f"  预期收益: {results[1, max_sharpe_idx]*252:.2%}")
print(f"  年化波动率: {results[0, max_sharpe_idx]*np.sqrt(252):.2%}")
print(f"  夏普比率: {results[2, max_sharpe_idx]:.3f}")

print(f"\n最小方差组合:")
print(f"  权重: Momentum={min_vol_weights[0]:.3f}, Mean_Reversion={min_vol_weights[1]:.3f}, Pairs={min_vol_weights[2]:.3f}")
print(f"  预期收益: {results[1, min_vol_idx]*252:.2%}")
print(f"  年化波动率: {results[0, min_vol_idx]*np.sqrt(252):.2%}")
```

## 使用SciPy优化找到精确的有效前沿

```python
def get_efficient_frontier(mean_returns, cov_matrix, n_points=100):
    """计算有效前沿"""
    n_assets = len(mean_returns)

    # 目标收益范围
    min_ret = mean_returns.min()
    max_ret = mean_returns.max()

    efficient_portfolios = []

    for target_return in np.linspace(min_ret, max_ret, n_points):
        # 约束条件
        constraints = [
            {'type': 'eq', 'fun': lambda x: np.sum(x) - 1},  # 权重和为1
            {'type': 'eq', 'fun': lambda x, t=target_return:
             portfolio_return(x, mean_returns) - t}  # 达到目标收益
        ]

        # 边界条件（每个权重0-100%）
        bounds = tuple((0, 1) for _ in range(n_assets))

        # 优化
        result = minimize(
            minimize_volatility,
            np.ones(n_assets) / n_assets,
            args=(mean_returns, cov_matrix),
            method='SLSQP',
            bounds=bounds,
            constraints=constraints
        )

        if result.success:
            efficient_portfolios.append({
                'weights': result.x,
                'return': target_return,
                'volatility': result.fun
            })

    return efficient_portfolios

efficient_fronteir = get_efficient_frontier(mean_returns, cov_matrix)
```

## 实际应用注意事项

### 1. 相关性不稳定

市场条件变化时，资产间的相关性会发生变化。2008年金融危机期间，许多"不相关"资产的实际相关性急剧上升。

### 2. 估计误差

均值-方差优化对输入参数非常敏感（"垃圾进，垃圾出"）。可以使用收缩估计器改善。

### 3. 交易成本

频繁调仓会产生大量交易成本，需要在优化中考虑。

### 4. 流动性限制

实际中不能无限买入小市值股票，需要考虑流动性约束。

from pypfopt.efficient_frontier import EfficientFrontier

# 计算预期收益和协方差
mu = expected_returns.mean_historical_return(prices)
Sigma = risk_models.sample_cov(prices)

ef = EfficientFrontier(mu, Sigma)

## 总结

| 概念       | 说明                                           |
| ---------- | ---------------------------------------------- |
| MPT        | 通过组合不同资产降低整体风险                   |
| 有效前沿   | 给定风险下收益最高，或给定收益下风险最低的组合 |
| 协方差矩阵 | 描述资产间共同运动规律的矩阵                   |
| 夏普比率   | 风险调整后的收益指标                           |
