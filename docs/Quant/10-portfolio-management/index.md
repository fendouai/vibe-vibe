# 第十章：组合管理

## 本章内容

本章我们将学习如何将多个策略组合在一起，构建一个稳健的投资组合。我们会探讨现代投资组合理论、多策略组合、凯利公式与杠杆应用，以及仓位再平衡策略。

## 学习目标

学完本章后，你将能够：

- 理解马科维茨投资组合理论的核心概念
- 掌握多策略组合的构建方法
- 应用凯利公式计算最优仓位
- 实现自动化的仓位再平衡
