# 5.1 动量效应原理：为何历史收益能预测未来

## 什么是动量效应

```python
"""
动量效应 (Momentum Effect)

定义：过去收益率较高的资产，在未来一段时间内
     仍将保持较高的收益率；
     过去收益率较低的资产，在未来一段时间内
     仍将保持较低的收益率。

简单说：强者恒强，弱者恒弱
"""

momentum_definition = """
研究历史：
- Jegadeesh & Titman (1993) 首次系统研究
- 股票市场中存在显著动量效应
- 持续期：3-12个月
- 反转期：12个月以上
"""
```

## 动量效应的解释

### 1. 行为金融学解释

```python
behavioral_explanations = {
    'anchoring': '锚定效应 - 投资者对新信息反应不足',
    'disposition': '处置效应 - 倾向于卖出盈利股、持有亏损股',
    'herding': '羊群效应 - 跟随大众行为',
    'confirmation': '确认偏差 - 只关注支持自己观点的信息'
}
```

### 2. 风险解释

```python
risk_explanations = {
    'priced_risk': '动量可能是对系统性风险的补偿',
    'skill': '部分基金经理的选股能力',
    'industry': '行业轮动效应'
}
```

## 动量周期

```python
"""
动量生命周期：

短期反转 ( < 1个月)
     ↓
中期动量 (1-12个月) ← 核心动量效应
     ↓
长期反转 ( > 12个月)

时间尺度     效应类型
< 1周       短期反转
1周-1月     弱反转/中性
1-12月      显著动量
12-36月     反转
> 36月      弱动量
"""
```

## 动量策略的类型

```python
# 1. 时间序列动量 (Time Series Momentum, TSMOM)
# 策略：做多过去收益高的资产，做空过去收益低的资产
# 参照： Moskowitz, Ooi, Pedersen (2012)

# 2. 横截面动量 (Cross Sectional Momentum, CSMOM)
# 策略：在同一资产类别中，做多收益高的，做空收益低的
# 参照： Jegadeesh & Titman (1993)

# 3. 因子动量 (Factor Momentum)
# 策略：做多动量因子强的资产组合
```
