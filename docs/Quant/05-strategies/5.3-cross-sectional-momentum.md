# 5.3 横截面动量：多资产组合、因子暴露

## 横截面动量原理

```python
"""
横截面动量 (CSMOM) 策略：

1. 计算多个资产过去N日的收益率
2. 按收益率排序
3. 做多收益率最高的资产，做空收益率最低的资产
4. 等权重持有

核心思想：相对强弱
"""
```

## Python 实现

```python
def cross_sectional_momentum(stocks_data, lookback=20, holding=10, n_long=1, n_short=1):
    """
    横截面动量策略

    Args:
        stocks_data: 多只股票的DataFrame
        lookback: 回看期
        holding: 持有期
        n_long: 做多数量
        n_short: 做空数量
    """
    # 计算动量（过去N日收益）
    momentum = stocks_data.pct_change(lookback)

    # 每holding天重新计算信号
    rebalance_dates = momentum.resample(f'{holding}d').last().dropna()

    signals = pd.DataFrame(0, index=rebalance_dates.index, columns=momentum.columns)

    for date in rebalance_dates.index:
        # 获取当日的动量排序
        mom = momentum.loc[:date].iloc[-1]
        ranked = mom.sort_values(ascending=False)

        # 做多前n_long个，做空后n_short个
        for i, ticker in enumerate(ranked.index):
            if i < n_long:
                signals.loc[date, ticker] = 1
            elif i >= len(ranked) - n_short:
                signals.loc[date, ticker] = -1

    # 前向填充到下一个调仓日
    signals = signals.reindex(momentum.index).ffill()
    signals = signals.shift(1).fillna(0)

    return signals

# 示例
stocks = ['AAPL', 'GOOGL', 'MSFT', 'AMZN', 'NVDA']
data = yf.download(stocks, start='2022-01-01', end='2023-12-31')['Close']
data.columns = data.columns.get_level_values(0)

signals = cross_sectional_momentum(data, lookback=60, holding=20, n_long=2, n_short=2)
```
