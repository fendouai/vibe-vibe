# 8.1 回测框架：向量化 vs 事件驱动

## 两种回测框架

```python
"""
1. 向量化回测 (Vectorized Backtesting)
   - 简单快速
   - 适合初步筛选策略
   - 无法处理复杂逻辑

2. 事件驱动回测 (Event-Driven Backtesting)
   - 真实模拟交易
   - 支持复杂策略
   - 计算量大
"""
```

## 向量化回测

```python
import pandas as pd
import numpy as np
import yfinance as yf

def vectorized_backtest(prices, signals):
    """
    向量化回测

    优点：
    - 简单快速
    - 用pandas广播一次性计算所有收益
    """
    # 计算收益率
    returns = prices.pct_change()

    # 策略收益 = 信号 * 收益
    # 信号移位避免前视偏差
    strategy_returns = signals.shift(1) * returns

    return strategy_returns

# 示例
df = yf.download('AAPL', start='2022-01-01', end='2023-12-31')
df.columns = df.columns.get_level_values(0)
prices = df['close']

# 简单动量信号
signals = (prices > prices.rolling(20).mean()).astype(int)

# 回测
returns = vectorized_backtest(prices, signals)
print(f"向量化回测总收益: {(1 + returns.dropna()).prod() - 1:.2%}")
```

## 事件驱动回测

```python
class EventDrivenBacktester:
    """
    事件驱动回测器

    优点：
    - 精确模拟交易
    - 支持订单类型、滑点等
    - 可以处理复杂策略逻辑
    """

    def __init__(self, initial_capital=100000):
        self.initial_capital = initial_capital
        self.cash = initial_capital
        self.position = 0
        self.trades = []
        self.portfolio_value = []

    def on_bar(self, date, price, signal):
        """每个bar调用一次"""
        if signal == 1 and self.position == 0:  # 买入
            shares = self.cash // price
            if shares > 0:
                self.cash -= shares * price
                self.position = shares
                self.trades.append({'date': date, 'type': 'BUY', 'price': price, 'shares': shares})

        elif signal == -1 and self.position > 0:  # 卖出
            self.cash += self.position * price
            self.trades.append({'date': date, 'type': 'SELL', 'price': price, 'shares': self.position})
            self.position = 0

        # 记录组合价值
        portfolio_value = self.cash + self.position * price
        self.portfolio_value.append({'date': date, 'value': portfolio_value})

    def run(self, prices, signals):
        """运行回测"""
        for date in prices.index:
            price = prices.loc[date]
            signal = signals.loc[date] if date in signals.index else 0
            self.on_bar(date, price, signal)

        return pd.DataFrame(self.portfolio_value).set_index('date')
```
