# 8.3 过拟合陷阱：数据窥视、前视偏差

## 回测的常见陷阱

```python
"""
回测的致命陷阱：

1. 前视偏差 (Look-ahead Bias)
   - 使用了未来才能获取的数据
   - 最常见的回测错误

2. 数据窥视 (Data Snooping)
   - 反复测试直到找到好结果
   - 导致过度优化

3. 幸存者偏差 (Survivorship Bias)
   - 只用现存股票测试
   - 高估策略表现

4. 流动性陷阱
   - 大资金无法按回测价格成交
"""
```

## 前视偏差

```python
"""
前视偏差示例：

错误：
- 用收盘价在收盘时下单
- 实际：收盘价只能在收盘时成交

正确：
- 用收盘价在开盘时下单（第二天开盘价成交）
- 或用下一根K线的开盘价成交
"""

# 正确处理：信号要shift
signals = some_strategy(prices)
signals = signals.shift(1)  # 避免前视偏差
strategy_returns = signals * prices.pct_change()
```

## 数据窥视

```python
"""
数据窥视 (Data Snooping)：

问题：
- 在同一数据集上反复测试
- 最终找到"恰好"有效的参数
- 但在新数据上完全失效

解决：
1. 样本外测试
2. Walk Forward 分析
3. 简化参数
4. 交叉验证
"""

# 示例：简单参数搜索（可能导致过拟合）
def optimize_parameters(prices):
    """简单的参数优化（可能过拟合）"""
    best_sharpe = -999
    best_params = None

    for fast in range(5, 50, 5):
        for slow in range(20, 200, 10):
            signals = (prices.rolling(fast).mean() > prices.rolling(slow).mean()).astype(int)
            returns = signals.shift(1) * prices.pct_change()
            sharpe = sharpe_ratio(returns.dropna())

            if sharpe > best_sharpe:
                best_sharpe = sharpe
                best_params = (fast, slow)

    return best_params, best_sharpe
```

## Walk Forward 分析

```python
def walk_forward_analysis(prices, train_window=252, test_window=63, fast_range=range(5, 50, 5), slow_range=range(20, 200, 10)):
    """
    Walk Forward 分析

    步骤：
    1. 用历史数据优化参数
    2. 在未见过的数据上测试
    3. 滚动窗口重复
    """
    results = []

    for i in range(train_window, len(prices) - test_window, test_window):
        # 训练集
        train_prices = prices.iloc[i-train_window:i]

        # 找最优参数
        best_sharpe = -999
        best_params = None

        for fast in fast_range:
            for slow in slow_range:
                if fast >= slow:
                    continue
                signals = (train_prices.rolling(fast).mean() > train_prices.rolling(slow).mean()).astype(int)
                returns = signals.shift(1) * train_prices.pct_change()
                sharpe = sharpe_ratio(returns.dropna())

                if sharpe > best_sharpe:
                    best_sharpe = sharpe
                    best_params = (fast, slow)

        # 测试集
        test_prices = prices.iloc[i:i+test_window]
        fast, slow = best_params

        test_signals = (test_prices.rolling(fast).mean() > test_prices.rolling(slow).mean()).astype(int)
        test_returns = test_signals.shift(1) * test_prices.pct_change()
        test_sharpe = sharpe_ratio(test_returns.dropna())

        results.append({
            'train_period': f"{prices.index[i-train_window].date()} to {prices.index[i-1].date()}",
            'test_period': f"{test_prices.index[0].date()} to {test_prices.index[-1].date()}",
            'train_sharpe': best_sharpe,
            'test_sharpe': test_sharpe,
            'fast': fast,
            'slow': slow
        })

    return pd.DataFrame(results)
```
