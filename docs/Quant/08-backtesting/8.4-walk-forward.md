# 8.4 Walk Forward：滚动窗口验证

## Walk Forward 原理

```python
"""
Walk Forward 分析：

是验证策略鲁棒性的金标准

步骤：
1. 定义训练窗口（如252天）
2. 定义测试窗口（如63天）
3. 用训练数据优化参数
4. 用测试数据验证
5. 滚动到下一期

关键指标：
- 训练集夏普 vs 测试集夏普的衰减
- 衰减 < 50% 通常可接受
"""
```

## Python 实现

```python
def walk_forward_optimization(prices, strategy_func, param_grid,
                               train_window=252, test_window=63):
    """
    Walk Forward 优化和验证

    Args:
        prices: 价格序列
        strategy_func: 策略函数，接受prices和参数，返回信号
        param_grid: 参数网格
        train_window: 训练窗口天数
        test_window: 测试窗口天数
    """
    results = []

    # 获取所有参数组合
    from itertools import product
    param_combinations = list(product(*param_grid.values()))
    param_names = list(param_grid.keys())

    for i in range(train_window, len(prices) - test_window, test_window):
        train_prices = prices.iloc[i-train_window:i]
        test_prices = prices.iloc[i:i+test_window]

        # 在训练集上找最优参数
        best_score = -np.inf
        best_params = None

        for params in param_combinations:
            param_dict = dict(zip(param_names, params))
            signals = strategy_func(train_prices, **param_dict)
            returns = signals.shift(1) * train_prices.pct_change()
            returns = returns.dropna()

            if len(returns) == 0:
                continue

            # 使用夏普比率作为评分
            score = sharpe_ratio(returns)

            if score > best_score:
                best_score = score
                best_params = param_dict

        # 在测试集上验证
        if best_params is not None:
            test_signals = strategy_func(test_prices, **best_params)
            test_returns = test_signals.shift(1) * test_prices.pct_change()
            test_returns = test_returns.dropna()

            if len(test_returns) > 0:
                test_sharpe = sharpe_ratio(test_returns)

                results.append({
                    'train_sharpe': best_score,
                    'test_sharpe': test_sharpe,
                    'decay': (best_score - test_sharpe) / best_score if best_score != 0 else 0,
                    **best_params
                })

    return pd.DataFrame(results)

def moving_average_crossover(prices, fast=20, slow=50):
    """移动平均交叉策略"""
    fast_ma = prices.rolling(fast).mean()
    slow_ma = prices.rolling(slow).mean()
    signals = (fast_ma > slow_ma).astype(int)
    return signals
```
