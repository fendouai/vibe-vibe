# 第 7 章：配对交易策略 (Pairs Trading)

## 配对交易原理

```python
"""
配对交易 (Pairs Trading)：

是一种市场中性策略，同时做多和做空两个高度相关的资产。

核心思想：
1. 找到一对高度相关的资产（A和B）
2. 当两者价差偏离历史均值时
3. 买入相对便宜的，卖空相对贵的
4. 等待价差回归，平仓获利

优势：
- 市场中性（与市场方向无关）
- 有限的下行风险
- 可以对冲系统性风险
"""

pairs_trading_principle = """
        价差(Spread)
            ↑
        均值 ─ ─ ─ ─ ─ ─ → 回归，平仓获利
            │    ↑
            │    │
            │    └─ 偏离加大
            │
            │    ↓
            └───────── 偏离加大
        └──────────────────→ 时间
"""
```
