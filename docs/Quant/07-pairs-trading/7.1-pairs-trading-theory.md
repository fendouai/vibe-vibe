# 7.1 配对交易原理：市场中性策略

## 市场中性策略

```python
"""
市场中性 (Market Neutral)：

策略收益与市场涨跌无关

实现方式：
- 同时做多和做空
- 组合的beta接近零
- 只关注相对价值变化

配对交易是最简单的市场中性策略
"""

market_neutral = """
市场中性收益来源：

总收益 = 多头收益 + 空头收益 + 价差收益
       = α × (β_market + β_stock) + ...

配对交易收益 ≈ 价差变化
             ≈ 价差的均值回归
"""
```

## 配对交易的数学基础

```python
"""
配对交易的数学：

假设两资产价格A和B满足：
A_t = β × B_t + ε_t

其中：
- β 是对冲比率
- ε_t 是价差（spread）

交易规则：
- 当 ε_t 偏离均值时
- 做多便宜的资产，做空贵的资产
- 等待 ε_t 回归
"""
```
