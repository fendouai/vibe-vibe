# 2.3 K线数据：OHLCV 数据结构与含义

## OHLCV 数据是什么

```python
# OHLCV 是金融市场的基本行情数据
ohlcv = {
    'O': 'Open（开盘价）',      # 周期开始时的第一个成交价
    'H': 'High（最高价）',      # 周期内的最高成交价
    'L': 'Low（最低价）',      # 周期内的最低成交价
    'C': 'Close（收盘价）',     # 周期结束时的最后一个成交价
    'V': 'Volume（成交量）'     # 周期内的总成交量
}
```

## 时间周期

```python
# 常见的时间周期
timeframes = {
    # 分钟级
    '1m': '1分钟K线',
    '5m': '5分钟K线',
    '15m': '15分钟K线',
    '30m': '30分钟K线',

    # 小时级
    '1h': '1小时K线',
    '4h': '4小时K线',

    # 日级
    '1d': '日K线',
    '1w': '周K线',
    '1M': '月K线',

    # 非标准
    'tick': '逐笔数据（每一笔成交）'
}
```

## OHLCV 数据处理

### 基础处理

```python
import pandas as pd
import yfinance as yf

df = yf.download('AAPL', start='2023-01-01', end='2023-12-31')
df.columns = df.columns.get_level_values(0)  # 处理列名

# 重命名列（统一为小写）
df.columns = [c.lower() for c in df.columns]
print(df.columns)  # ['open', 'high', 'low', 'close', 'adj close', 'volume']

# 选取需要的列
df = df[['open', 'high', 'low', 'close', 'volume']]
```

### 计算典型价格

```python
# 典型价格（Typical Price）= (H + L + C) / 3
df['typical_price'] = (df['high'] + df['low'] + df['close']) / 3

# 加权收盘价（Weighted Close）= (H + L + 2*C) / 4
df['weighted_close'] = (df['high'] + df['low'] + 2 * df['close']) / 4

# 中点价 = (H + L) / 2
df['mid_price'] = (df['high'] + df['low']) / 2
```

### 计算收益率

```python
# 简单收益率
df['return'] = df['close'].pct_change()

# 对数收益率（更适合统计分析）
import numpy as np
df['log_return'] = np.log(df['close'] / df['close'].shift(1))

# 多日收益率
df['return_5d'] = df['close'].pct_change(5)   # 5日收益率
df['return_20d'] = df['close'].pct_change(20)  # 20日收益率

# 年化收益率
trading_days = 252
df['annualized_return'] = df['return'].mean() * trading_days
df['annualized_vol'] = df['return'].std() * np.sqrt(trading_days)
```

## 常用技术指标

### 移动平均线 (MA)

```python
# 简单移动平均 (SMA)
df['sma_20'] = df['close'].rolling(window=20).mean()
df['sma_60'] = df['close'].rolling(window=60).mean()

# 指数移动平均 (EMA)
df['ema_12'] = df['close'].ewm(span=12).mean()
df['ema_26'] = df['close'].ewm(span=26).mean()

# 移动平均交叉信号
df['ma_signal'] = 0
df.loc[df['sma_20'] > df['sma_60'], 'ma_signal'] = 1   # 金叉
df.loc[df['sma_20'] < df['sma_60'], 'ma_signal'] = -1  # 死叉
```

### 波动率指标

```python
# 历史波动率（年化）
df['hist_vol'] = df['return'].rolling(20).std() * np.sqrt(252)

# ATR (Average True Range)
df['tr'] = np.maximum(
    df['high'] - df['low'],
    np.maximum(
        abs(df['high'] - df['close'].shift(1)),
        abs(df['low'] - df['close'].shift(1))
    )
)
df['atr_14'] = df['tr'].rolling(14).mean()
```

### RSI (Relative Strength Index)

```python
def calculate_rsi(prices, period=14):
    delta = prices.diff()
    gain = (delta.where(delta > 0, 0)).rolling(period).mean()
    loss = (-delta.where(delta < 0, 0)).rolling(period).mean()
    rs = gain / loss
    rsi = 100 - (100 / (1 + rs))
    return rsi

df['rsi_14'] = calculate_rsi(df['close'], 14)
```

## 数据质量检查

```python
# 检查缺失值
print(df.isnull().sum())

# 检查异常值
print(df.describe())

# 检查价格是否为0
print(df[df['close'] == 0])

# 检查成交量是否为0（可能意味着停牌）
print(df[df['volume'] == 0])

# 检查价格跳变（可能意味着数据问题）
df['price_change'] = df['close'].pct_change()
print(df[abs(df['price_change']) > 0.5])  # 日涨跌超过50%的异常数据
```
