## 核心概念

### 金融工具的风险收益特征

| 工具     | 潜在收益 | 潜在风险       | 适合策略   |
| -------- | -------- | -------------- | ---------- |
| 股票     | 无上限   | 100%（本金）   | 动量、价值 |
| 期货     | 无上限   | 无上限（杠杆） | 趋势、套利 |
| 期权     | 无上限   | 有限（期权费） | 波动率交易 |
| 数字货币 | 无上限   | 100%           | 趋势、套利 |
