# 2.1 金融工具：股票、期货、期权、数字货币

## 主要金融工具概览

量化交易可以在多种金融工具上执行。每种工具有其独特的特性和策略。

## 股票 (Stocks)

### 基本概念

```python
# 股票代表上市公司所有权的一部分
# 买入股票 = 成为公司股东

# 股票代码示例
stocks = {
    'AAPL': 'Apple Inc.',
    'GOOGL': 'Alphabet Inc.',
    'MSFT': 'Microsoft Corporation',
    '600519': '贵州茅台 (A股)',
    '000858': '五粮液 (A股)'
}
```

### 特点

| 特性     | 说明                     |
| -------- | ------------------------ |
| 收益来源 | 资本增值 + 分红          |
| 风险     | 相对有限（最大亏损100%） |
| 交易时间 | 交易所规定时段           |
| 结算周期 | T+1 或 T+2               |

### 量化常用指标

```python
# 市值 (Market Cap)
market_cap = stock_price * shares_outstanding

# 市盈率 (P/E Ratio)
pe_ratio = stock_price / earnings_per_share

# 市净率 (P/B Ratio)
pb_ratio = stock_price / book_value_per_share

# 股息率 (Dividend Yield)
dividend_yield = annual_dividend / stock_price
```

## 期货 (Futures)

### 基本概念

```python
# 期货是约定在未来某一时间以特定价格买卖资产的合约
# 期货代码示例
futures = {
    'ES': 'E-mini S&P 500 期货',
    'NQ': 'E-mini Nasdaq-100 期货',
    'CL': 'WTI 原油期货',
    'GC': '黄金期货',
    'IF': '沪深300股指期货 (A股)'
}
```

### 特点

| 特性     | 说明                 |
| -------- | -------------------- |
| 杠杆     | 高杠杆（通常 5-10%） |
| 保证金   | 开仓需缴纳保证金     |
| 到期日   | 有固定到期日，需展期 |
| 结算方式 | 大多数现金结算       |

### 期货的优势

```python
# 1. 高杠杆 - 用较少资金控制大额资产
# 假设保证金率 5%，则 20倍杠杆

# 2. 做空容易 - 可以方便做空
# 股票做空需要借券，期货直接卖出即可

# 3. 交易成本低 - 期货交易费用通常较低
# 4. 流动性好 - 主力合约流动性极佳
```

## 期权 (Options)

### 基本概念

```python
# 期权赋予持有者在特定时间以特定价格买入/卖出资产的权利
# 期权代码示例
options = {
    'AAPL230331C180': 'AAPL 2023-03-31 到期 执行价180 看涨期权',
    'SPY230630P400': 'SPY 2023-06-30 到期 执行价400 看跌期权'
}

# Call Option (看涨期权) = 买入的权利
# Put Option (看跌期权) = 卖出的权利
```

### 期权关键参数

```python
# 期权价格 = 内在价值 + 时间价值
# Black-Scholes 模型定价

# 希腊字母 (Greeks) - 衡量风险
greeks = {
    'Delta': '价格对标的价格变化的敏感度',  # dPrice/dUnderlying
    'Gamma': 'Delta 对标的价格变化的敏感度',  # dDelta/dUnderlying
    'Theta': '价格对时间流逝的敏感度',        # dPrice/dTime
    'Vega': '价格对波动率的敏感度',           # dPrice/dVolatility
    'Rho': '价格对利率的敏感度'              # dPrice/dRate
}
```

### 期权策略

```python
# 1. 保护性看跌 (Protective Put)
# 持有股票 + 买入看跌期权 = 下行保护
protective_put = {
    'position': 'long_stock + long_put',
    'max_loss': 'put_strike - stock_price + premium',
    'upside': 'unlimited'
}

# 2. 备兑看涨 (Covered Call)
# 持有股票 + 卖出看涨期权 = 增加收益
covered_call = {
    'position': 'long_stock + short_call',
    'max_loss': 'stock_price - premium',
    'upside': 'capped_at_strike'
}

# 3. 跨式期权 (Straddle)
# 同时买入看涨和看跌期权
straddle = {
    'position': 'long_call + long_put (same strike)',
    'profit_if': 'large_move_in_either_direction',
    'max_loss': 'call_premium + put_premium'
}
```

## 数字货币 (Cryptocurrencies)

### 基本概念

```python
# 数字货币是去中心化的加密资产
crypto = {
    'BTC': 'Bitcoin',
    'ETH': 'Ethereum',
    'BNB': 'Binance Coin',
    'SOL': 'Solana'
}

# 7x24 交易，无休
# 交易所：币安、Binance、OKX、Coinbase
```

### 特点

| 特性     | 说明        |
| -------- | ----------- |
| 交易时间 | 7x24 小时   |
| 波动性   | 高波动性    |
| 杠杆     | 可高达 100x |
| 监管     | 监管较少    |

### 数字货币特有的量化策略

```python
# 1. 跨交易所套利
# 由于交易所之间价格可能存在差异
# 低价交易所买入，高价交易所卖出

# 2. 资金费率套利 (Funding Rate Arbitrage)
# 利用永续合约的资金费率

# 3. 网格交易
# 在震荡行情中设置网格自动化买卖
```

## 金融工具选择

### 根据策略类型选择

| 策略类型   | 推荐工具       | 原因                   |
| ---------- | -------------- | ---------------------- |
| 趋势跟随   | 期货、数字货币 | 高杠杆、可以做空       |
| 均值回归   | 股票、期权     | 波动有限、可构建保护   |
| 配对交易   | 股票、期货     | 便于做空、交易成本低   |
| 波动率交易 | 期权、数字货币 | 波动率高、期权策略丰富 |

### 新手推荐

**建议从股票开始**，因为：

- 相对简单
- 风险有限
- 流动性好
- 信息透明
