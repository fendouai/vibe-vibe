## 订单簿 (Order Book)

### 订单簿结构

```python
# 订单簿显示所有未成交的买卖订单
# 这是市场的"供需表"

order_book = {
    'bids': [  # 买单（想要买入）
        {'price': 99.95, 'size': 500},   # 最高买入价
        {'price': 99.90, 'size': 1000},
        {'price': 99.85, 'size': 2500},
    ],
    'asks': [  # 卖单（想要卖出）
        {'price': 100.05, 'size': 700},   # 最低卖出价
        {'price': 100.10, 'size': 1200},
        {'price': 100.15, 'size': 800},
    ]
}

# 当前最佳买卖价
best_bid = order_book['bids'][0]['price']  # 99.95
best_ask = order_book['asks'][0]['price']  # 100.05
mid_price = (best_bid + best_ask) / 2     # 100.00
spread = best_ask - best_bid              # 0.10
```

### 订单簿可视化

```python
import matplotlib.pyplot as plt

def plot_order_book(order_book):
    """可视化订单簿"""
    fig, ax = plt.subplots(figsize=(10, 6))

    bids = order_book['bids']
    asks = order_book['asks']

    # 绘制深度

    ask_prices = [a['price'] for a in asks]
    ask_sizes = [a['size'] for a in asks]
    ask_cumsum = np.cumsum(ask_sizes)

    ax.barh(bid_prices, bid_cumsum, label='Bids (Buy)', color='green', alpha=0.6)

    ax.set_xlabel('Cumulative Size')
    ax.set_ylabel('Price')
    ax.set_title('Order Book Depth')
    ax.legend()
    plt.show()
```

## 市场价格形成

### 订单匹配逻辑

```python
# 交易所如何匹配订单？

match_logic = """
1. 价格优先：买入订单按出价从高到低成交
             卖出订单按出价从低到高成交

2. 时间优先：同一价格，先提交的订单先成交

3. 举例：
   - 买入订单：100@99.95（100股，价格99.95）
   - 卖出订单：100@100.00（100股，价格100.00）
   - 结果：无法成交（买入价<卖出价）

   - 买入订单：100@100.05
   - 卖出订单：100@100.00
   - 结果：以100.00成交（按卖出方价格）
"""
```

### 市场订单 vs 限价订单

```python
# 市场订单 (Market Order)
market_order = {
    'type': 'market',
```
