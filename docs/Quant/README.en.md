<h1 align="center">VibeQuant —— AI-Powered Quantitative Trading for Everyone</h1>

> **A systematic tutorial on AI-assisted quantitative trading for beginners, transforming "I have a trading idea" into "I built a strategy" - enabling everyone to realize their quantitative dreams with AI.**

### Core Philosophy

Based on the **Vibe Coding** concept by Andrej Karpathy: **From Coder to Commander** - using natural language to converse with AI, transforming programming from "writing code" to "conversational creation."

> _"Immerse yourself in the 'vibe' of quantitative trading, forget the existence of code."_

## Technology Stack

- **Language**: Python 3.10+
- **Data**: pandas, numpy, yfinance, akshare
- **Strategies**: ta-lib, statsmodels, scipy
- **Backtesting**: backtrader, vectorbt, custom frameworks
- **Live Trading**: alpaca-trade-api, IB API, Binance API

## Each Chapter Includes

- 3-5 curated reference resources (free courses, books, open source projects)
- Hands-on Python implementations
- Complete working code examples
- Practice exercises with solutions

## Learning Path (6-8 weeks for beginners)

```
Week 1-2: Python basics + environment setup
Week 3: Financial market basics
Week 4: Data acquisition and processing
Week 5: Financial data analysis
Week 6-7: Classic strategies (momentum, mean reversion)
Week 8: Backtesting methodology
```
