# 0.4 环境搭建：Python 环境配置与必备库

## 选择你的开发环境

### 选项一：Anaconda（推荐零基础）

Anaconda 是一个打包好的 Python 发行版，预装了大部分科学计算所需的库。

**下载地址**：[https://www.anaconda.com/download](https://www.anaconda.com/download)

**优点**：

- 开箱即用
- 包含 Jupyter Notebook
- 包含大部分常用库

**缺点**：

- 体积较大（约 800MB）
- 部分库版本可能较旧

### 选项二：Miniconda + pip（推荐有经验者）

Miniconda 是 Anaconda 的轻量版，只包含 Python 和 conda。

```bash
# 安装 Miniconda
# macOS/Linux:
curl -O https://repo.anaconda.com/miniconda/Miniconda3-latest-MacOSX-arm64.sh
bash Miniconda3-latest-MacOSX-arm64.sh

# Windows: 从 https://docs.conda.io/en/latest/miniconda.html 下载安装
```

### 选项三：纯 Python + pip（不推荐零基础）

如果你已经有 Python 环境，可以直接用 pip 安装：

```bash
pip install jupyterlab numpy pandas matplotlib seaborn yfinance akshare
```

## 创建虚拟环境

**强烈建议**：为量化交易创建独立的虚拟环境。

```bash
# 使用 conda 创建环境
conda create -n quant python=3.11
conda activate quant

# 使用 venv 创建环境
python -m venv quant_env
source quant_env/bin/activate  # macOS/Linux
quant_env\Scripts\activate   # Windows
```

## 安装必备库

### 核心库

```bash
# 数据处理
pip install numpy pandas

# 可视化
pip install matplotlib seaborn plotly

# 金融数据获取
pip install yfinance          # Yahoo Finance 美股数据
pip install akshare          # A股、期货等多品种数据
pip install baostock         # 免费A股数据

# 策略技术指标
pip install ta ta-lib        # 技术分析指标

# 统计建模
pip install statsmodels scipy

# 回测框架
pip install backtrader       # 经典回测框架
pip install vectorbt         # 现代化回测框架（基于pandas）

# 机器学习（后续章节使用）
pip install scikit-learn tensorflow
```

### 如果安装 ta-lib 失败

ta-lib 在某些系统上安装可能较复杂，可以先使用 ta（纯 Python 实现）：

```bash
pip install ta
# ta 提供了大部分 ta-lib 的功能
```

## Jupyter Notebook / JupyterLab

Jupyter 是量化交易开发的标准工具。

```bash
# 安装 JupyterLab（推荐，功能更强）
pip install jupyterlab

# 或者安装经典 Jupyter Notebook
pip install notebook

# 启动
jupyter lab
# 或
jupyter notebook
```

## 验证安装

在 Jupyter 中运行以下代码验证安装：

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import yfinance as yf

# 测试获取数据
data = yf.download('AAPL', start='2023-01-01', end='2023-12-31')
print(f"获取到 {len(data)} 条数据")
print(data.tail())
```

如果能正常输出数据，说明环境配置成功。

## 开发工具推荐

### VS Code + Jupyter 插件（推荐）

1. 安装 [VS Code](https://code.visualstudio.com/)
2. 安装 Jupyter 插件
3. 打开 .ipynb 文件即可开发

### PyCharm

功能完整的 Python IDE，适合大型项目。

### Google Colab（云端，无需安装）

如果不想在本地配置，可以使用 Google Colab：

- 免费使用 GPU/TPU
- 预装大部分库
- 网址：https://colab.research.google.com/

## 项目文件夹结构

建议创建如下文件夹结构：

```
quant_project/
├── data/               # 存放数据文件
│   ├── raw/           # 原始数据
│   └── processed/     # 处理后数据
├── strategies/         # 策略代码
│   ├── momentum.py
│   ├── mean_reversion.py
│   └── pairs_trading.py
├── backtests/         # 回测结果
├── notebooks/         # Jupyter notebooks
├── logs/              # 日志文件
└── requirements.txt   # 依赖列表
```

## requirements.txt 示例

```txt
numpy>=1.24.0
pandas>=2.0.0

seaborn>=0.12.0
```

创建 requirements.txt：

```bash
pip freeze > requirements.txt
```
