## 先把边界说清

第 0 章的任务很简单：先把量化交易的核心概念说清楚，把学习路径理顺，也把一些常见的误解消除。

很多人第一次听到"量化交易"时，最常见的反应是：

- "这是不是只有金融专业人士才能学的？"
- "我数学不好，能学会吗？"
- "量化交易是不是就是高频交易？"

第 0 章先把这些问题说清楚。

看完这一章，你至少会更确定两件事：量化交易到底是怎么回事，以及你需不需要学它。

## 先记住一句话

> 量化交易不是"写代码代替人决策"，而是"用系统化方法把人的交易想法变成可执行的规则"。

量化交易的核心是**系统化**——把模糊的交易想法，变成清晰的、可以回测验证的、可以规模化执行的规则。
