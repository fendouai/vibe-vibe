# 第十一章：实盘交易

## 本章内容

本章我们将学习如何将策略连接到真实市场进行交易。我们会介绍主流的券商API（Alpaca、Interactive Brokers、Binance），以及实盘交易的核心概念和注意事项。

## 学习目标

学完本章后，你将能够：

- 理解券商API的工作原理
- 使用Alpaca API进行美股交易
- 使用Binance API进行数字货币交易
- 实现从回测到实盘的无缝切换
