# 第十三章：高级机器学习

## 本章内容

本章我们将学习如何将机器学习和深度学习应用于量化交易。从特征工程、机器学习模型，到模型验证和深度学习，我们将探索如何利用现代AI技术提升交易策略的效果。

## 学习目标

学完本章后，你将能够：

- 理解量化交易中的特征工程方法
- 掌握常用机器学习模型的原理和应用
- 正确验证和测试ML模型
- 了解深度学习在交易中的应用
- 避免ML模型过拟合的陷阱
