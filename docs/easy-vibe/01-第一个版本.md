# 第 1 章：第一个版本 - 2 小时做出个人主页 + 数字分身

---

## 本章目标

**先做出一个能预览、能展示、能简单聊天的 v1 原型**

- 不需要学会完整提示词理论
- 不需要一次把作品做到完美
- **最重要的：先把第一个作品做出来**

---

## 第一步：准备素材（5 分钟）

### 不用从零写 prompt，先收集事实

| 类型     | 记录什么                         |
| -------- | -------------------------------- |
| 基本信息 | 名字、头像、一句话介绍           |
| 当前状态 | 最近在做什么、最想让别人记住什么 |
| 兴趣特点 | 2-4 个兴趣点、一个有记忆点的特点 |
| 访客视角 | 可能来访问你主页的人是谁         |
| 常见问题 | 别人最可能问你的 3 个问题        |

### 好素材的标准

❌ 太空的写法

- 我是一个有热情的人
- 我喜欢很多东西

✅ 更有用的写法

- 我最近在整理自己的作品，也在尝试用 AI 做个人项目
- 我比较关注 AI 应用、写作和旅行

**记住**：具体比抽象更有用，真实比好听更有用

---

## 第二步：用模板写需求（10 分钟）

### 复制并填空

```text
我想做一个个人主页，上面有一个数字分身聊天区。

【关于我】
- 名字：______
- 一句话介绍：______
- 我现在主要在做：______
- 我的兴趣：______
- 我一个比较有记忆点的特点：______

【这个主页给谁看】
- 主要访客：______

【页面里至少要有】
- 头像
- 名字和一句话介绍
- 一个简单的个人信息展示区
- 一个可以提问的数字分身聊天区

【数字分身需要知道】
- 我的职业或身份：______
- 我最近在做的事：______
- 我擅长或关心的方向：______
- 别人最可能问我的 3 个问题：______

【风格要求】
- 我希望整体感觉是：______
- 我更偏好的颜色：______
- 是否要适配手机：是
- 其他要求：______

请先帮我生成一个第一版原型。
要求：
1. 先保证结构清楚、能用
2. 不要过度复杂
3. 先做出一个可预览的版本
```

### 填好的示例

```text
【关于我】
- 名字：林安
- 一句话介绍：一个正在学习用 AI 做产品的内容策划
- 我现在主要在做：整理自己的作品和写作方向
- 我的兴趣：AI 应用、写作、旅行
- 我一个比较有记忆点的特点：喜欢把复杂问题讲成人话

【这个主页给谁看】
- 主要访客：朋友、潜在合作伙伴、面试官

【数字分身需要知道】
- 我的职业或身份：内容策划
- 我最近在做的事：搭自己的个人主页，整理作品集
- 我擅长或关心的方向：内容表达、AI 应用、知识整理
- 别人最可能问我的 3 个问题：你现在在做什么？你有哪些作品？怎么联系你？

【风格要求】
- 我希望整体感觉是：简约、清爽、稍微有一点科技感
- 我更偏好的颜色：深蓝色和白色
- 是否要适配手机：是
```

## 第三步：在平台生成（1 小时）

### 操作顺序

1. 新建项目，命名如 `个人主页-v1`
2. 粘贴填好的模板（不要一边贴一边改）
3. 等待第一轮输出结束（不要频繁打断）
4. 进入预览页

### 第一次预览只看三件事

- ✅ 页面结构是否出来
- ✅ 个人信息是否基本正确
- ✅ 数字分身聊天区是否出现

**第一页有问题很正常**，比如：

- 名字或信息抽错
- 聊天框位置生硬
- 链接还不能跳
- 按钮只是占位

**记住**：第一个版本的目标不是"惊艳"，而是"先出来一个能继续改的版本"。

---

## 第四步：三轮微调（30 分钟）

### 微调原则

1. **一次只改一类问题** - 不要一口气改配色、布局、聊天逻辑
2. **把改动边界说清** - 明确说"只改这一部分"、"不要新增模块"

### 第 1 轮：修明显错误

只处理最明显的问题：

- 名字写错、信息抽错
- 区块缺失、布局严重不对
- 聊天入口没有正常出现
- 链接按钮不能跳

```
请先不要大改整个页面。

我先修几个明显问题：
1. 我的名字显示不对，请改成正确版本
2. 右侧聊天框离页面右边缘太近，请给更自然的外边距
3. 页面里的外部链接请接成真实可点击跳转
4. Q1/Q2/Q3 如果只是演示按钮，要么改成真实触发，要么先移除

这一轮先只修这些明显问题，不要重做整体风格。
```

### 第 2 轮：调风格和内容

处理"能用但不像我"的问题：

- 风格太花或太素
- 颜色不一致
- 信息层次不够清楚

```
现在结构已经基本对了。

接下来我只想调整整体风格：
- 让页面更简约、清爽
- 主色调改成深蓝和白色
- 不要增加新的复杂模块

请只做风格层面的修改。
```

### 第 3 轮：小幅细化

处理"不是大问题，但做完会更顺眼"的细节：

- 按钮文案更自然
- 聊天区提示语更清楚
- 头像和简介排版更舒服

```
页面整体已经接近我想要的效果了。

现在请只做小幅细化：
- 把聊天框的引导语改得更自然一点
- 让头像和简介区的间距更舒服
- 不要新增模块，也不要重做布局
```

### 哪些现在修，哪些后面修

| 现在就修               | 后面几章处理              |
| ---------------------- | ------------------------- |
| 名字写错、信息抽错     | 界面和风格统一（第 3 章） |
| 明显布局问题、区块缺失 | 内容模块补全（第 4 章）   |
| 链接按钮不能跳转       | 数字分身更像你（第 5 章） |
| 聊天区只是占位         | 上线与真实环境（第 6 章） |

## 本章验收清单

### 过关标准

- [ ] 页面能正常打开
- [ ] 我的名字和一句话介绍清楚显示
- [ ] 页面看起来像"我的个人主页"而不是空白模板
- [ ] 聊天入口存在，并且能完成基础提问
- [ ] 手机端至少可以正常浏览

### 检查动作

**假装自己是第一次访问的朋友，问自己三个问题：**

1. 我能不能一眼知道这是谁？
2. 我知不知道应该点哪里、问什么？
3. 这个页面现在值不值得继续迭代？
