# 第 3 章：做好第一印象 - 界面、风格与需求表达

---

## 本章目标

**让首页从默认模板变成更像认真做过的作品**

- 选一个清晰的风格方向
- 抓住最值回票价的三个界面改动
- 学会把"我想要它更..."说清楚

---

## 第一步：先选风格方向（5 分钟）

### 为什么先改界面

- **反馈最快** - 改完立刻能看出来
- **最直观** - 一眼就能判断好还是不好
- **练表达能力** - 把模糊感觉说清楚

### 风格方向不要多，先求统一

三种可选方向：

| 方向       | 更像什么感觉                 | 适合什么场景                      |
| ---------- | ---------------------------- | --------------------------------- |
| 简约专业   | 克制、清楚、信息优先         | 想让别人快速看懂你是谁、做什么    |
| 温暖友好   | 有人味、愿意交流、距离感更低 | 想强调表达、内容、沟通感          |
| 轻微科技感 | 清爽、数字感、稍微有未来感   | 本来就在做 AI、产品、技术相关内容 |

### 写一句方向描述

```text
我想要一个简约专业、信息层次清楚、带一点轻微科技感的个人主页。
```

或者：

```text
我希望首页看起来温暖友好、容易交流，
不要太像简历，也不要太像模板站。
```

**关键**：方向要统一，不要一会想要简约专业，一会想要温暖友好

---

## 第二步：三个最值回票价的改动（30 分钟）

### 1. 先把首屏排版调顺

首屏决定别人第一眼看到什么。

**重点检查四个点**：

- 名字和一句话介绍是不是足够显眼
- 头像是不是在帮信息服务，不是抢信息
- 主按钮是不是能一眼看出要做什么
- 数字分身入口是不是存在感太弱

**让 AI 帮你改**：

```text
请先优化首页首屏排版。

目标是让名字、一句话介绍和主要按钮更有主次，
聊天入口更容易被注意到。

不要新增模块，先只调整布局、间距和元素优先级。
```

### 2. 颜色和字体先求统一

**重点不是玩花样，而是统一**

| 先定什么 | 够用标准                                |
| -------- | --------------------------------------- |
| 主色     | 只保留 1 个主色，不要每个按钮都一个颜色 |
| 背景气质 | 明确整体偏干净明亮，还是偏稳重克制      |
| 字体层次 | 标题和正文要有区别，不要像一整块灰字    |

**目标**：让页面看起来像做过选择，而不是默认值的拼接

### 3. 区块层次与留白

决定页面是不是好读。

**直接检查**：

- 各区块之间是不是挤成一团
- 标题、正文、辅助说明有没有明显层次
- 页面是不是一眼能扫出"首屏、内容区、聊天区"这几个区域

## 第三步：把感觉说清楚（15 分钟）

### 有效的说法，有三层

```text
现在的问题是什么
  ↓
我想改成什么感觉
  ↓
这次只动哪些地方，不动哪些地方
```

### 示例

```text
现在这个页面首屏太像默认模板，信息有点散，
聊天入口也不够明显。

我希望它更简洁、更有层次，
更像一个认真做过的个人主页。

请先只调整首屏排版、颜色和间距，
不要新增模块，也不要改聊天逻辑。
```

### 也可以给 AI 一个参考物

参考物可以是：

- 一张你现在页面的截图
- 一个你觉得风格不错的网站
- 一句很短的风格描述

```text
我想让它更接近"简洁、留白多、信息层次清楚"的感觉，
不要太像简历站，也不要太像炫技 landing page。
```

### 好看之外，还要好用

顺手把体验要求说进去：

- 首屏要一眼知道我是谁
- 聊天入口要更明显
- 手机端要保证信息可读
- 按钮要有基本反馈

### 可直接复用的模板

```text
现在这个页面的主要问题是：______

我希望它改成这样的感觉：______

这次请你只修改这些地方：______
先不要动这些部分：______

另外请兼顾这些最基本的体验：
- 首屏一眼能看出我是谁
- 聊天入口更明显
- 手机端文字可读
- 不要让页面显得更乱
```
