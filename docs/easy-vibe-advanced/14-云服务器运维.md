## 安全加固

### 为什么需要

裸奔的服务器在互联网上，平均几分钟就会被扫描到

### 必做项

- 更新系统
- 配置安全组

## Docker 容器

### 什么是 Docker

**集装箱**——每个应用独立运行，互不干扰

### 核心概念

| 概念         | 说明         |
| ------------ | ------------ |
| **容器**     | 运行中的应用 |
| **镜像**     | 容器的模板   |
| **端口映射** | 外部访问容器 |

---

## 容器网络

### 关键规则

| 场景       | 使用什么       |
| ---------- | -------------- |
| 容器找容器 | 用容器名       |
| 主机找容器 | `localhost`    |
| 外部访问   | 公网 IP + 端口 |

**容器里的 `localhost` 是容器自己，不是服务器主机**

## 日志

### 看什么

| 关键词         | 含义           |
| -------------- | -------------- |
| **Error**      | 错误——定位问题 |
| **Warning**    | 警告——值得检查 |
| **你写的日志** | 执行到了哪一步 |

### 常见模式

| 日志信息     | 原因               |
| ------------ | ------------------ |
| "找不到模块" | 依赖没装好         |
| "连接被拒绝" | 地址错或没启动     |
| "超时"       | 网络问题或逻辑问题 |

## SRE 入门意识

### 什么是 SRE

**站点可靠性工程**——用软件工程思维解决运维问题

### 基础能力

- 监控——了解系统状态
- 自动化——一键部署、一键回滚
- 容灾——备份和恢复
