## localhost 问题

### 场景

复制 `http://localhost:3000` 发给朋友，朋友打不开

### 原因

**localhost 只有你自己能访问**

朋友访问的是他自己的电脑，他电脑上没有运行你的项目

---

## 网络三个层级

| 层级                  | 范围               | 示例                  |
| --------------------- | ------------------ | --------------------- |
| **本地（localhost）** | 只有你自己         | `127.0.0.1:3000`      |
| **局域网**            | 同一个 WiFi 下的人 | `192.168.1.5:3000`    |
| **公网**              | 所有人             | `https://example.com` |

---

## 局域网真机调试

### 如何做

1. 确保手机和电脑在同一个 WiFi
2. 查看电脑局域网 IP（如 `192.168.1.5`）
3. 手机浏览器访问 `192.168.1.5:3000`

### 可能的问题

- 防火墙拦截端口
- 路由器隔离设备

---

## 内网穿透

### 什么是内网穿透

**临时让朋友看到你的本地项目**

### 工具

**Cloudflare Tunnel** —— 免费、简单、安全

### 使用方法

```bash
# 安装
brew install cloudflare/cloudflare/cloudflared  # Mac

# 运行
cloudflared tunnel --url http://localhost:3000
```

### 特点

- 关了终端链接就失效
- 不需要公网 IP
- 安全（不暴露真实地址）

---

## 为什么需要部署

**部署 = 让项目永久在线**

Tunnel 是临时方案，部署是永久方案
