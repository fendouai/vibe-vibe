## SEO 基础配置

### Metadata

在 Next.js 的 `layout.tsx` 配置：

```typescript

  title: '你的网站名称',
  description: '网站描述'
}
```

## Open Graph 分享

### 问题

链接发到群里，只显示一串蓝色链接，没有标题或图片

### 解决方案

**配置 OG 协议**

在 Next.js 中：

- 放一张图片命名为 `opengraph-image.png`
- 自动生成 OG 标签
- 社交平台抓取时显示精美大图

## Umami 数据统计

### 为什么选择 Umami

- 轻量、隐私友好
- 可以自己搭建（用 1Panel）

### 使用方法

1. 在 1Panel 一键部署 Umami
2. 把生成的 JS 代码贴到应用
3. 后台查看访问数据

## 法律合规

### 必做项

| 要求         | 说明                     |
| ------------ | ------------------------ |
| **隐私政策** | 说明数据用途、存储方式   |
| **用户协议** | 服务条款、免责声明       |
| **ICP 备案** | 服务器在中国大陆必须备案 |

**合规是产品长期运营的基石**
