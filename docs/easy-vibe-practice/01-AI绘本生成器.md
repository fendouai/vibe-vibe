## 第一阶段：Demo（5 分钟）

### Step 1：下载 AI 编辑器（2 分钟）

- 搜索 **"Cursor 下载"** 或 **"Windsurf 下载"**
- 安装并注册账号

### Step 2：创建项目

新建文件夹：`我的AI绘本`

### Step 3：复制这段话给 AI

```
我想做一个 AI 绘本生成器。

请帮我创建一个最简单的版本：
1. 用 Python + Gradio 做界面
2. 用户输入一段文字描述
3. 调用 AI 生成一张绘本风格的图片
4. 显示出来

先帮我创建项目结构和必要的文件。
```

### Step 4：接入 Nano Banana Pro API

1. 打开 [https://grsai.com](https://grsai.com)
2. 注册账号并创建 API Key
3. 创建 `.env` 文件：

```bash
NANO_BANANA_API_KEY=你的API密钥
NANO_BANANA_API_BASE=https://grsai.com
NANO_BANANA_MODEL=nano-banana-pro
```

### Step 5：运行看效果

```bash
pip install gradio openai
python app.py
```

打开浏览器访问 `http://localhost:7860`

## 第三阶段：优化

### 关键技巧

| 技巧                             | 说明                     |
| -------------------------------- | ------------------------ |
| **先描述"做什么"，再说"怎么做"** | 不要一开始就说技术细节   |
| **描述现象而非猜测原因**         | 遇到问题要描述具体表现   |
| **用具体例子说明期望**           | 不要说"不够好"，要给例子 |
| **分步骤迭代**                   | 不要一次说太多需求       |

### 踩坑指南

| 坑                       | 解决方案                            |
| ------------------------ | ----------------------------------- |
| **API Key 泄露**         | 写在 `.env` 文件，加入 `.gitignore` |
| **图片风格不统一**       | 每个 Prompt 都加风格描述            |
| **角色长得不一样**       | 先生成角色参考图，每次生成时传入    |
| **生成太慢**             | 加进度条，实时显示每张图            |
| **Prompt 太长，AI 忘记** | 重要要求放在开头和结尾              |

## Prompt 模板

### 大纲生成

```
你是一位世界级的儿童绘本设计师。

## 任务
根据文档内容，设计一本 10-15 页的绘本大纲。

## 输出要求
为每一页提供：
1. 标题（叙事性标题）
2. 叙事目标
3. 关键内容
4. 视觉画面
5. 布局建议

## 禁止
- 禁止"让我们一起..."等说教语气
- 禁止"不仅仅是X，而是Y"句式
```

### 图片生成

```
为绘本的第 {页码} 页绘制插图。

## 风格要求
- 迪士尼皮克斯 3D 动画风格
- 明亮温暖的色调
- 16:9 比例，4K 分辨率

## 角色一致性
主角必须和参考图保持一致
```

## 第四阶段：架构理解

### 系统流程

```
用户上传文档
    ↓
文档解析（MinerU OCR）
    ↓
AI 生成绘本大纲
    ↓
生成角色参考图
    ↓
批量生成每页插图
    ↓
导出 PPT/PDF
```

### 技术栈

| 层           | 技术                     |
| ------------ | ------------------------ |
| **后端**     | Python Flask             |
| **数据库**   | SQLite                   |
| **文档解析** | MinerU OCR               |
| **AI 模型**  | GPT-4o + Nano Banana Pro |
