# 核心概念与范式演进

## 编者按

我们正站在软件开发史上的一个转折点。

当 AI 从"辅助工具"变成"协作伙伴"，当自然语言成为新的编程接口，当开发者的角色从"代码编写者"转变为"系统设计者"——这不是渐进式的改良，而是范式级的革命。

Vibe Coding 和 Agentic Engineering 不是流行词汇，而是对这场革命的命名。它们重新定义了"什么是编程"这个根本问题：编程不再是逐行编写代码，而是与智能体协作，通过自然语言表达意图，让 AI 理解上下文并生成解决方案。

本专栏收录的文章来自 Vercel 和 Anthropic 的一线工程师——他们不是在谈论未来，而是在分享现在。这些文章回答了最本质的问题：

- Vibe Coding 的核心是什么？不是"让 AI 写代码"，而是建立人机协作的新范式
- 智能体工程师需要什么能力？不是更强的编码技巧，而是上下文管理和任务分解
- 这场变革的边界在哪里？哪些是真正的突破，哪些是过度炒作

如果你想理解这场变革的本质，而不是追逐表面的工具和技巧，这里是起点。
