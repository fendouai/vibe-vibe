# 第十三章：域名、DNS 与网络接入

![img](/images/Advanced/mll08c63-bcea35d79958ea70.jpg)

## 序言

小明的应用终于部署成功了。

他迫不及待地把 EdgeOne Pages 分配的默认链接发到群里："快来看我做的网站！"

三分钟后，朋友截图回复：**403 Forbidden**。

小明慌了——"我明明部署成功了啊？"他反复刷新自己的页面，一切正常。但朋友那边就是打不开。

老师傅看了一眼截图，笑了："别慌。403 就是'门关着，你没钥匙'。国内平台的默认链接有访问限制，只有你自己能看。想让别人也能访问，得绑一个属于你自己的域名。"

"域名？就是那个 `.com` 的东西？"

"对。域名就是互联网的通讯录——你在通讯录里搜'妈妈'，手机帮你拨号。浏览器搜 `myapp.com`，DNS 帮你找到服务器。你现在用的是平台的临时号码，别人查不到。得注册一个正式的号码，写进通讯录里。"

小明想起网上有免费的二级域名可以领——就是那种 `你的名字.别人的域名.com`，挂在别人域名下面的。老师傅严肃地拦住了他："那是借别人家的门牌号，房东随时能把你踢走。做产品，域名就是门面。花几十块钱买个正经的 `.com`，值得。"

这一章，小明要给自己的网站起个正式的名字。
