# 使用 AI Gateway 生成视频

> **作者**：Jerilyn Zheng  
> **来源**：https://vercel.com/blog/video-generation-with-ai-gateway  
> **核心观点**：AI SDK 6 提供统一的视频生成接口

---

## 两种入门方式

| 方式                      | 说明                             |
| ------------------------- | -------------------------------- |
| **AI SDK 6**              | 编程式生成，与文本/图片相同接口  |
| **AI Gateway Playground** | 无代码实验，直接在模型详情页试用 |

---

## 四个初始视频模型

| 模型                    | 特点                               |
| ----------------------- | ---------------------------------- |
| **Grok Imagine（xAI）** | 速度快、指令遵循强、风格转换       |
| **Wan（阿里巴巴）**     | 参考生成、多镜头叙事、角色一致性   |
| **Kling（快手）**       | 图生视频、原生音频、多镜头场景转换 |
| **Veo（Google）**       | 高视觉保真、物理真实感、电影感画面 |

---

## 四种生成类型

| 类型             | 输入                | 用例                       |
| ---------------- | ------------------- | -------------------------- |
| **文本生成视频** | 文本提示词          | 广告创意、解说视频         |
| **图片生成视频** | 图片 + 可选文本     | 产品展示、Logo动画         |
| **首尾帧**       | 两张图片 + 可选文本 | 前后对比、转场效果         |
| **参考生成视频** | 图片或视频          | 品牌角色内容、统一形象系列 |

---

## 代码示例

### 文本生成视频

```typescript
import { experimental_generateVideo as generateVideo } from 'ai'

const { videos } = await generateVideo({
  model: 'klingai/kling-v2.6-t2v',
  prompt: '火箭从发射台升起...',
  aspectRatio: '16:9',
  duration: 5,
  providerOptions: { klingai: { mode: 'pro', sound: 'on' } }
})
```

### 图片生成视频

```typescript
const { videos } = await generateVideo({
  model: 'klingai/kling-v2.6-i2v',
  prompt: {
    image: blackHoodie,
    text: '猫咪慢慢走过黑色卫衣...'
  },
  duration: 5
})
```

### 参考生成视频

```typescript
const { videos } = await generateVideo({
  model: 'alibaba/wan-v2.6-r2v-flash',
  prompt: 'character1 和 character2 在海滩玩耍...',
  providerOptions: {
    alibaba: { referenceUrls: [shibaImage, yorkieImage] }
  }
})
```

## 模型能力对比

| 模型创建方 | 能力                                           |
| ---------- | ---------------------------------------------- |
| xAI        | 文本生成视频、图片生成视频、视频编辑、音频     |
| Wan        | 文本生成视频、图片生成视频、参考生成视频、音频 |
| Kling      | 文本生成视频、图片生成视频、首尾帧、音频       |
| Veo        | 文本生成视频、图片生成视频、音频               |

---

## 关键结论

> **你可以用与文本和图片相同的接口生成电影级视频，统一的 API、统一的鉴权、统一的可观测性。**
